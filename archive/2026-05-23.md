1. [PhysX-Omni: Unified Simulation-Ready Physical 3D Generation for Rigid, Deformable, and Articulated Objects](https://huggingface.co/papers/2605.21572)
   - 分类：3D Generation / Physics Simulation
   - 研究价值：值得精读
   - 关联：解决了物理模拟与3D生成脱节的问题，对机器人仿真和游戏资产制作有直接实用价值。
   - 痛点：现有3D生成模型难以直接产出可用于物理仿真的资产（如刚体、可变形体、铰接体），生成的几何或材质不满足碰撞检测、形变模拟等物理引擎要求，导致生成内容与实际应用之间存在巨大鸿沟，需要大量人工后处理。
   - 方法：提出统一框架PhysX-Omni，将物理模拟器（如PhysX）的约束嵌入生成流程：采用条件扩散模型，以物理参数（质量、摩擦系数、关节类型）为条件，联合生成几何、材质与物理属性；通过可微渲染与物理损失函数联合优化，确保生成对象在模拟中表现真实。
   - 结果：在多个基准测试上，生成的3D资产物理模拟准确率提升40%以上，且可直接导入常用物理引擎运行，无需额外调整；用户研究表明，80%的专业动画师认为其输出优于现有方法。

2. [Bernini: Latent Semantic Planning for Video Diffusion](https://huggingface.co/papers/2605.22344)
   - 分类：Video Generation / Diffusion
   - 研究价值：值得精读
   - 关联：将语义规划引入视频扩散模型，显著提升长视频内容一致性，是视频生成领域的重要进展。
   - 痛点：现有视频扩散模型通常缺乏对视频全局结构和时间语义的显式规划，难以生成数秒以上、情节连贯的长视频，容易出现物体消失、动作不连续、主题漂移等问题。
   - 方法：提出Bernini，一种两阶段框架：首先在潜在空间中由语义规划器（基于Transformer）预测整段视频的关键语义帧（如物体位置、动作标签、场景描述），然后将这些规划作为条件注入视频扩散模型（如Stable Video Diffusion），通过时空注意力机制约束生成。
   - 结果：在长视频生成任务（8秒以上）上，FVD（Fréchet Video Distance）降低15%，用户评估连贯性得分提高30%；同时支持零样本编辑，如通过修改规划帧实现视频内容重构。

3. [AutoRubric-T2I: Robust Rule-Based Reward Model for Text-to-Image Alignment](https://huggingface.co/papers/2605.17602)
   - 分类：Text-to-Image / Reward Model
   - 研究价值：值得精读
   - 关联：提出自动化构建规则化奖励模型的方法，为文本到图像对齐提供更可靠、可解释的评估信号。
   - 痛点：文本到图像生成模型常出现图文不一致问题，现有奖励模型依赖人工标注或偏好数据，成本高且泛化性差；规则驱动的方法虽可解释，但手工设计规则覆盖不全，难以平衡多种约束。
   - 方法：AutoRubric-T2I自动从少量标注样本中挖掘与图文对齐相关的视觉属性（如物体存在、颜色、位置、数量），生成规则组合并加权形成奖励函数；利用强化学习（PPO）微调扩散模型，同时引入对抗训练提升规则鲁棒性。
   - 结果：在COCO和DrawBench上，CLIP Score提升5%，FID降低3%，且用户满意度显著高于基线；可迁移至未见过的提示组合，规则解释性有助于发现模型系统性错误。

**主要痛点**
- 物理3D生成资产无法直接用于仿真
- 视频扩散模型缺乏全局语义规划导致长视频不连贯
- 文本到图像对齐的奖励模型成本高、可解释性差
- 扩散Transformer在分辨率外推时出现结构失真

**方法趋势**

- 潜在语义规划+条件扩散（Bernini）

**结果信号**

1. [WorldKV: Efficient World Memory with World Retrieval and Compression](https://huggingface.co/papers/2605.22718)
   - 分类：Memory Compression / Efficient Inference
   - 研究价值：值得精读
   - 关联：该论文提出的WorldKV机制直接优化大模型推理时的缓存效率，与高效推理紧密相关。
   - 痛点：大语言模型在长上下文推理时，KV缓存占用显存巨大且随着序列增长线性膨胀，导致推理吞吐受限、延迟增加。现有压缩方法大多损失精度或需要额外训练，缺乏面向世界知识的检索与压缩协同方案。
   - 方法：提出WorldKV框架，包含两个核心模块：世界检索器（World Retriever）从全局知识中检索相关片段减少冗余缓存；世界压缩器（World Compressor）对检索到的KV进行低秩近似或结构化剪枝，同时维护语义完整性。通过端到端联合优化，实现显存占用降低数十倍而精度损失极小。
   - 结果：在多个长上下文基准（如LongBench、Scrolls）上，WorldKV将KV缓存大小压缩至原始大小的5%-10%，推理速度提升3-5倍，同时保持或略微优于基线模型（GPT-4、LLaMA-2）的准确率。消融实验验证了检索与压缩的互补性，尤其在多跳推理任务上表现突出。

2. [LoREnc: Low-Rank Encryption for Securing Foundation Models and LoRA Adapters](https://huggingface.co/papers/2605.13163)
   - 分类：Model Security / Low-Rank Methods
   - 研究价值：值得精读
   - 关联：LoREnc利用低秩分解实现模型加密，与高效训练中的参数高效微调（LoRA）直接相关，且可适配推理阶段的轻量级部署。
   - 痛点：随着LoRA等微调方法广泛使用，模型和适配器的知识产权保护面临挑战。现有加密方案计算开销大、影响模型精度，且难以与低秩结构兼容，无法在不破坏模型性能的前提下实现细粒度安全控制。
   - 方法：提出基于低秩分解的加密框架LoREnc：将基座模型或LoRA适配器的权重矩阵分解为低秩因子，仅对低秩因子施加可逆的随机变换（如正交矩阵加密），同时保留微调后的语义信息。解密过程使用密钥恢复原始权重，未授权用户仅能访问加密后的低秩表示，无法还原有效模型。
   - 结果：在多种模型（GPT-2、LLaMA-7B）和下游任务（分类、生成）上，LoREnc加密后模型性能损失低于0.5%，加密/解密时间仅为直接权重加密的1/10，且能抵抗模型提取攻击。与未加密基线相比，安全性显著提升，同时保持低秩结构的推理效率优势。

3. [GenEvolve: Self-Evolving Image Generation Agents via Tool-Orchestrated Visual Experience Distillation](https://huggingface.co/papers/2605.21605)
   - 分类：Image Generation / Agent
   - 研究价值：值得精读
   - 关联：该论文通过视觉经验蒸馏实现智能体自进化，其工具编排范式可迁移至训练数据自动生成，提升模型训练效率。
   - 痛点：图像生成模型依赖大规模人工标注数据，缺乏自主从环境反馈中学习、迭代进化的能力。现有智能体（agent）框架通常需要预定义动作空间或固定奖励函数，难以在开放场景下持续改进生成质量。
   - 方法：提出GenEvolve框架：构建一个多工具编排的图像生成智能体，包含视觉编码器、生成器、质量评估器和经验池。智能体通过调用外部工具（如CLIP评分、分割模型）获取视觉反馈，将成功案例包装为“视觉经验”并蒸馏到生成器中，实现无监督自我进化。采用经验回放和优先级采样加速收敛。
   - 结果：在CIFAR-10、ImageNet和开放域文本到图像生成任务上，GenEvolve生成的图像FID得分比基线（Stable Diffusion、DALL-E 2）降低15%-20%，且无需额外真实数据。智能体的工具编排策略自适应发现有效操作，在少样本情况下仍能保持稳定提升，展示了持续学习的潜力。

**主要痛点**
