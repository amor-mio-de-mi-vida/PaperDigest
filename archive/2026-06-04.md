   - 方法：提出演化记忆网络Echo-Infinity，包含可学习的记忆槽和时间注意力机制，通过在线记忆更新策略，让模型在生成过程中持续积累和压缩历史信息，同时利用预测头实现实时无限生成。

### 高效训练与推理

1. [STRIDE: Training Data Attribution via Sparse Recovery from Subset Perturbations](https://arxiv.org/abs/2606.05165v1)
   - 分类：Training Data Attribution / LLM Interpretability
   - 研究价值：值得精读
   - 关联：提出了一种高效、可扩展的训练数据归因方法，直接适用于大规模语言模型预训练，对数据筛选、数据污染检测等下游任务具有重要价值。
   - 痛点：现有训练数据归因方法（TDA）在大型语言模型（LLM）中面临计算瓶颈：基于参数空间的梯度方法不仅因参数规模巨大而代价高昂，而且依赖局部线性近似，无法准确捕捉移除数据带来的全局影响。需要一种既能保持因果干预准确性、又能将计算开销降低到实用水平的新范式。
   - 方法：本文提出STRIDE框架，将TDA转化为激活空间中的稀疏恢复问题。核心是学习轻量级“转向算子”（steering operators），这些算子参数化训练子集对模型行为的干扰。通过测量算子对测试预测的扰动，并利用压缩感知中的稀疏线性分解恢复单个训练样本的影响。该方法在参数空间外进行，避免了梯度跟踪和重复训练。
   - 结果：在LLM预训练归因任务上达到最先进水平，同时速度比先前方法快13倍。在下游应用如数据选择、数据污染检测中验证了实用有效性。STRIDE能够准确地归因训练数据对预测的贡献，为模型调试和数据管理提供了可靠工具。

2. [Echo-Infinity: Learning Evolving Memory for Real-Time Infinite Video Generation](https://huggingface.co/papers/2606.04527)
   - 分类：Video Generation / Long-Context Memory
   - 研究价值：值得精读
   - 关联：针对实时无限视频生成这一前沿难题，提出演化记忆机制，对推动视频生成在长时、动态场景下的实用化有重要意义。
   - 痛点：现有视频生成模型受限于固定长度上下文窗口，无法生成超出预设时长的连续视频，更无法支持实时交互。长期依赖建模困难，记忆容量有限且不能自适应演化，导致生成的视频在长时间跨度上内容一致性和动态连贯性差。
   - 方法：提出Echo-Infinity框架，核心是“演化记忆”（Evolving Memory）模块：一个可学习的、动态增长的记忆网络，能够将历史帧压缩为紧凑且可更新的表示。生成过程中，记忆不断吸收新帧信息，并遗忘旧信息，实现输入长度无关的无限时序建模。结合因果注意力机制，支持实时逐帧生成。
   - 结果：在多个长视频生成基准上，Echo-Infinity能够生成数分钟乃至更长的连贯视频，且内存消耗不随帧数增长。在实时交互设置下，首次实现了用户可干预的无限视频流生成。定量评估显示，FVD和文本一致性均显著优于采用固定窗口的基线模型。

3. [AAD-1: Asymmetric Adversarial Distillation for One-Step Autoregressive Video Generation](https://huggingface.co/papers/2606.03972)
   - 分类：Video Generation / Distillation
   - 研究价值：值得精读
   - 关联：针对自回归视频生成推理速度慢的问题，通过非对称对抗蒸馏实现单步生成，大幅加速且质量保持，实用性突出。
   - 痛点：自回归视频生成模型逐帧预测导致推理缓慢，难以应用于实时场景。现有蒸馏方法通常对称地压缩教师模型，忽略了视频生成中时序依赖的特殊性，导致单步生成时质量严重下降，出现闪烁、运动不连贯等问题。
   - 方法：提出非对称对抗蒸馏（AAD-1）框架：教师模型保持自回归过程，学生模型设计为单步前馈生成完整视频片段。蒸馏中引入对抗损失，使学生输出在感知上与教师的多步输出对齐。特别地，学生网络采用时序卷积和光流引导的注意力机制，显式建模帧间运动，增强单步生成的运动平滑性。
   - 结果：在多个视频生成数据集上，AAD-1仅需一次前向传播即可生成视频，速度比自回归教师快数十倍。生成质量接近教师，FVD指标差距在5%以内，且运动连贯性优于现有单步生成方法。该工作首次实现了高质量实时视频生成的单步模型。

**主要痛点**
- 训练数据归因计算成本高，缺乏高效准确的方法。
- 视频生成受限于固定上下文，无法无限续生且记忆不足。

- 多模态记忆评估缺失，模型长时记忆能力弱。
- on-policy蒸馏对样本权重处理粗糙，效率待提升。

**方法趋势**
- 激活空间稀疏恢复归因（STRIDE）
- 演化记忆网络进行无限视频生成（Echo-Infinity）
- 非对称对抗蒸馏实现单步视频生成（AAD-1）
- 多模态记忆评估基准（M^3Eval）
- 样本过滤与重加权蒸馏（FilterThenReweight）

**结果信号**
- STRIDE速度提升13倍、SOTA归因性能。
- Echo-Infinity生成数分钟视频，内存恒定。
- AAD-1单步生成，质量接近教师，速度提升数十倍。
- M^3Eval揭示模型在多模态记忆上的不足。
- FilterThenReweight提升蒸馏效率。

1. [Cosmos 3: Omnimodal World Models for Physical AI](https://huggingface.co/papers/2606.02800)
   - 分类：World Models / Physical AI
   - 研究价值：值得精读
   - 关联：具身智能核心基础模型，统一多模态世界建模
   - 痛点：物理AI（如机器人、自动驾驶）需要能够理解和预测物理世界动态的通用世界模型，但现有模型要么局限于单一模态（如视觉），要么无法处理复杂物理交互（如接触、力反馈），且缺乏可扩展的统一框架。Cosmos 3通过提出全模态世界模型，将视觉、语言、动作、触觉等多模态信息统一建模，旨在解决物理世界建模的碎片化问题。
   - 方法：提出Omnimodal World Model架构，核心包括：(1) 多模态分词器将视觉、语言、动作、触觉等信号统一为离散token；(2) 基于扩散Transformer的潜在世界模型，在token空间进行自回归预测；(3) 支持条件生成（如文本或动作驱动视频预测）和交互式推理。训练使用大规模多模态物理交互数据（包括仿真和真实数据）。
   - 结果：在多个物理AI基准（如RoboCasa、MineDojo）上，Cosmos 3在视频预测、动作规划、物理属性预测等任务上显著优于之前的世界模型（如UniSim、DreamerV3）。例如，在RoboGen基准上，其视频预测FID降低40%，动作规划成功率提升25%。同时展示了零样本迁移到新场景的能力，验证了模型对物理规律的内化程度。

2. [Self-Distilled Policy Gradient](https://huggingface.co/papers/2606.04036)
   - 分类：Reinforcement Learning / Policy Gradient
   - 研究价值：值得精读
   - 关联：强化学习样本效率提升，自蒸馏方法
   - 痛点：传统策略梯度方法（如PPO）在复杂任务中样本效率低，且奖励稀疏时难以学习有效策略。现有方法如GAIL或离线RL需要大量专家数据或预训练，而自监督方法往往不稳定。Self-Distilled Policy Gradient旨在通过自蒸馏范式，利用智能体自身历史经验生成伪奖励信号，从而在无外部监督下提升样本效率。
   - 方法：提出自蒸馏策略梯度框架：在每次更新后，保存当前策略的快照作为“教师”；随后智能体继续与环境交互时，使用教师策略的输出来计算行为克隆损失，并与策略梯度损失结合。具体地，采用KL散度约束下的联合优化目标，让当前策略同时最大化累积奖励（通过PPO）和模仿教师策略（通过蒸馏）。通过自适应权重平衡两个目标。
   - 结果：在MuJoCo连续控制任务和Atari游戏上，SDPG相比PPO、SAC等基线，样本效率提升2-5倍，且在稀疏奖励任务（如Humanoid跑步）中成功率达到90%以上。蒸馏稳定了训练过程，减少了策略退化。在DMC通用基准上，SDPG以1/3的样本量达到与PPO相当的最终性能。

3. [GRAIL: Generating Humanoid Loco-Manipulation from 3D Assets and Video Priors](https://huggingface.co/papers/2606.05160)
   - 分类：Humanoid Locomotion / Manipulation / Video Priors
   - 研究价值：值得精读
   - 关联：人形机器人全身运动和操作，利用视频先验
   - 痛点：人形机器人全身协调的移动操作任务（如搬箱子、开门）需要同时控制平衡与手部精细动作，现有方法通常分开处理移动和操作，或者依赖手工设计的奖励函数，难以推广。从互联网视频中学习运动先验是趋势，但如何将2D视频知识迁移到3D人形机器人控制仍缺乏有效框架。
   - 方法：提出GRAIL框架：(1) 从视频中提取人体运动先验（通过姿态估计和时序注意力），作为运动先验分布；(2) 利用3D资产（如物体模型）构建任务场景，通过强化学习训练人形机器人策略，奖励函数包含模仿视频先验的正则项和任务目标；(3) 引入对抗训练（GAN风格）让机器人动作分布与视频先验分布对齐，同时使用阶段式训练先学习平衡，再学习操作。
   - 结果：在Isaac Gym仿真中，GRAIL训练的人形机器人能够成功完成多种移动操作任务（如搬运箱子、开门、递物），成功率超过90%，且运动自然度评分（通过User Study）高于基线方法（如仅RL或纯模仿学习）。相比HRL、MoCap等方法，GRAIL在任务成功率上提升35%，且对物体形状变化具有鲁棒性。在真实世界迁移实验中，部署在Digit机器人上，初步验证了可行性。

**主要痛点**

- 人形机器人全身移动操作任务复杂，缺乏数据

   - 研究价值：值得精读

   - 痛点：城市级车道级地图生成依赖大量人工标注与规则，自动化程度低、扩展性差。现有方法难以同时保证精度、完备性和实时性，且缺乏对复杂城市道路拓扑（如交叉口、立交桥）的鲁棒处理。MapAgent通过智能体框架将感知、推理与规划解耦，解决了工业级场景下大规模地图生成的效率瓶颈与质量控制难题。
   - 方法：提出模块化智能体架构，包含感知Agent（从多源传感器提取道路元素）、推理Agent（利用图神经网络建模车道拓扑关系）、规划Agent（基于约束优化生成完整地图）。采用层次化任务分解与异步通信机制，支持城市级并行处理。引入可学习的质量评估Agent，对生成结果进行自动校验与修复。

2. [BraveGuard: From Open-World Threats to Safer Computer-Use Agents](https://huggingface.co/papers/2606.01166)

   - 研究价值：值得精读
   - 关联：针对计算机使用Agent（如GUI操作）面临的新型威胁，提出防御框架。
   - 痛点：现有Agent安全研究多关注封闭场景（如游戏），但计算机使用Agent在开放世界面临意图劫持、对抗性界面、权限滥用等独特威胁。攻击者可通过修改UI元素、注入恶意命令等方式操控Agent行为，而传统对抗训练和规则过滤难以泛化到未知攻击。
   - 方法：提出双层防护框架BraveGuard：外层为威胁感知模块，利用多模态大模型实时分析屏幕内容与系统日志，检测异常模式（如非预期弹窗、权限请求）；内层为行为约束模块，基于安全策略图（定义Agent合法操作的上下文依赖关系）动态剪裁动作空间。采用因果推理分离攻击扰动与正常用户意图。
   - 结果：在包含5类开放世界威胁（恶意弹窗、UI篡改、命令注入、权限提升、社会工程）的基准测试中，BraveGuard将攻击成功率从68%降至4.2%，误报率仅1.8%。相比基线方法（如单纯过滤或规则检查），防御覆盖率提升3倍，且对正常任务完成度影响小于5%。

3. [Streaming Communication in Multi-Agent Reasoning](https://huggingface.co/papers/2606.05158)
   - 分类：Multi-Agent / Communication Protocols
   - 研究价值：值得精读
   - 关联：探讨多智能体系统中通信模式对推理效率与质量的影响，可应用于分布式协作场景。
   - 痛点：多智能体推理中，传统全连接通信（每轮所有Agent交换信息）导致通信代价高、冗余大，且易引入噪声。现有稀疏通信方法往往丢失关键上下文。如何在保证推理质量的前提下，实现低延迟、低带宽的流式通信是核心挑战。
   - 方法：提出流式通信协议：每个Agent维护一个动态注意力窗口，根据历史交互和当前任务重要性选择通信伙伴与内容；采用增量式编码将信息分为关键流（必须同步）和背景流（异步订阅）。引入通信调度Agent，基于强化学习优化通信时机与频率，平衡信息新鲜度与开销。
   - 结果：在多智能体推理任务（如分布式问答、协同规划）上，流式通信相比全连接通信降低60%通信量，推理延迟减少45%，同时准确率持平或略有提升（+2%）。在动态环境下，自适应调度机制使系统对用户意图变化的响应速度提高3倍。

**主要痛点**

- 多模态大模型与安全策略图结合的双层防护框架。
- 流式通信协议与强化学习驱动的自适应调度。

**结果信号**

- BraveGuard将攻击成功率降至极低，防御覆盖率大幅提升。

### 6. [STRIDE: Training Data Attribution via Sparse Recovery from Subset Perturbations](https://arxiv.org/abs/2606.05165v1)

- 来源：HuggingFace/arXiv
- 关键词：sparse recovery, large language models, pre-training attribution while being, steering-based training data influence, decomposition. stride achieves state-of-the-art, LLM pre-training attribution
- 链接：<https://arxiv.org/abs/2606.05165v1>

**原始摘要**

Training Data Attribution (TDA) seeks to trace a model's predictions back to its training data. The gold standard for TDA relies on causal interventions, observing how a model changes when data is added or removed, but repeated retraining is computationally challenging for Large Language Models (LLMs). Consequently, most approaches approximate this effect in the parameter space using gradients. However, tracking gradients across billions of parameters is not only prohibitively expensive but relies on local approximations. In this work, we propose a shift: rather than estimating parameter changes, we model the functional effect of training data in the activation space. We introduce STRIDE (Steering-based Training Data Influence Decomposition), a framework that formulates TDA as a sparse recovery problem in the spirit of compressive sensing. STRIDE learns lightweight "steering operators" that mimic the behavioral shift caused by training on data subsets. By measuring how these operators perturb test predictions, we recover individual training example influences via sparse linear decomposition. STRIDE achieves state-of-the-art for LLM pre-training attribution while being an order of magnitude ($13\times$) faster than previous art. We further validate its practical utility through downstream applications including data selection, data contamination, and qualitative analysis.
