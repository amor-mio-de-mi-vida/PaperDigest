1. [ARM: An AutoRegressive Large Multimodal Model with Unified Discrete Representations](https://arxiv.org/abs/2606.11188v1)
   - 分类：Multimodal Generation / Autoregressive Models
   - 研究价值：值得精读
   - 关联：实现了多模态任务（理解、生成、编辑）的统一，为构建通用多模态AI提供了新范式。
   - 痛点：当前多模态AI模型通常将图像理解、生成和编辑分离为独立任务，缺乏统一的表示和架构，限制了跨任务知识迁移和泛化能力。离散视觉表示虽然有利于与语言模态对齐，但现有分词器在保持语义判别性的同时难以充分保留低级视觉细节，导致在生成任务中重建质量不足。此外，多任务联合训练时缺乏有效的偏好对齐机制，生成结果往往与人类期望不一致。
   - 方法：提出离散语义视觉分词器，通过多目标联合训练（语义判别性、语言对齐、忠实重建）获得紧凑且信息丰富的离散token。在此基础上构建7B参数的自回归模型，在共享潜在空间处理文本和图像token序列，无缝覆盖理解、生成和编辑。为进一步对齐人类偏好，在文本到图像生成和指令编辑任务上应用强化学习，以视觉质量、指令遵循和编辑一致性为目标进行优化。
   - 结果：经RL优化后，ARM在WISE基准上整体得分从0.50提高到0.56，在GEdit-Bench-EN的G_O指标从5.75提升至6.68，均达到当时最佳水平。更重要的是，RL训练带来了跨任务协同：文本到图像生成的质量提升同时改善了编辑任务的一致性。该工作证明了自回归框架在统一多模态任务上的潜力，为后续研究提供了strong baseline。

2. [IDEAL: In-DEpth ALignment Makes A Discrete Representation AutoEncoder](https://arxiv.org/abs/2606.11096v1)
   - 分类：Image Generation / Discrete Representation
   - 研究价值：值得精读
   - 关联：揭示了浅层特征在离散表示中的关键作用，大幅提升了自回归图像生成的质量。
   - 痛点：现有的表示自编码器依赖深度视觉基础模型特征构建潜在空间，但深层特征偏向语义而丢失大量低级视觉细节（如纹理、边缘），导致离散化后重建图像模糊、保真度低。这一问题在自回归生成中尤为严重，因为离散token的有限容量难以同时承载复杂外观和高级语义。现有方法忽视浅层特征在保留局部结构中的关键作用，缺乏跨层特征融合策略。
   - 方法：提出In-Depth ALignment（IDEAL）框架，在离散表示自编码器中联合对齐量化token与预训练视觉基础模型的浅层和深层特征。具体地，通过两个对齐分支分别从浅层（保留细节）和深层（保留语义）提取特征，并设计损失函数促使离散token包含互补信息。训练后，离散token同时具备视觉保真度和丰富语义。
   - 结果：在ImageNet 256x256上，IDEAL取得重建rFID 0.61，相比之前最佳方法（0.89）提升31%，达到当时SOTA。用于自回归图像生成时，gFID为1.89，同样创下新纪录。实验还表明，浅层特征的对齐是提升细节质量的关键，且模型具有良好的泛化能力。

3. [Next Forcing: Causal World Modeling with Multi-Chunk Prediction](https://arxiv.org/abs/2606.11187v1)
   - 分类：Video Generation / World Models
   - 研究价值：值得精读
   - 关联：将多token预测思想扩展到视频生成，显著加速训练和推理，为世界模型的高效学习提供了新思路。
   - 痛点：自回归视频生成模型通常只在当前时间步施加监督，缺乏对未来动态的显式信号，导致训练收敛缓慢，尤其在处理高帧率视频时。此外，推理阶段需要多步迭代去噪，计算成本高，难以满足实时应用需求。现有方法没有充分利用未来信息来加速训练和推理。
   - 方法：提出Next Forcing多块预测框架，在主干自回归模型上添加轻量级辅助MCP模块，同时预测未来多个时间步的视频块。MCP模块形成因果链：近未来预测为远未来提供中间特征。训练时提供密集的多尺度未来监督，加速收敛；推理时MCP模块可并行预测下一块，实现2倍加速。
   - 结果：在50fps高帧率下，Next Forcing在5000步训练时相比LingBot-VA相对提升93.1%，收敛速度快2.3倍。在RoboTwin基准上达到94.1% Clean和93.5% Random准确率，均创SOTA。推理速度达到2倍加速，且FVD在通用视频预训练上降低超50%。

**主要痛点**
- 多模态任务缺乏统一表示和架构；离散表示难以平衡语义与细节。
- 视频生成训练收敛慢，缺乏未来监督，推理迭代成本高。
- 生成模型与实际应用（实时、对齐、3D）之间存在差距。

**方法趋势**
- 自回归模型与离散表示结合，强化学习偏好优化。
- 多块预测框架，因果链结构，辅助模块加速训练推理。
- 蒸馏、测试时训练、替代轨迹优化等高效学习策略。

**结果信号**
- ARM统一多模态任务并展现跨任务协同；IDEAL大幅提升图像生成质量。
- Next Forcing实现2.3倍更快收敛和2倍推理加速；MFD单步生成SOTA。
- WorldOlympiad揭示世界模型在物理、3D、交互上的不足。

### Agent

- 论文数：11

**今日推荐精读 Top 3**

1. [EEVEE: Towards Test-time Prompt Learning in the Real World for Self-Improving Agents](https://arxiv.org/abs/2606.11182v1)
   - 分类：Test-time Prompt Learning / Self-Improving Agents
   - 研究价值：值得精读
   - 关联：首个解决真实世界多数据集测试时提示学习的框架，与TRACE互补了Agent自我改进方法。
   - 痛点：现有测试时提示学习方法仅支持单数据集设定，而真实应用需要处理来自多个数据集、领域和任务分布的异构输入流，导致实用性受限。跨数据集干扰使得单一提示无法适应所有任务，Agent在动态环境中难以持续自我改进。EEVEE针对这一痛点，首次提出面向多任务流的测试时提示学习，使Agent能在真实非平稳分布下自主适应，显著提升了鲁棒性和泛化能力。
   - 方法：EEVEE引入一个路由器模块，将输入划分为任务簇并分配对应的提示配置。为解决路由器与提示相互依赖的优化难题，提出路由-提示协同进化策略，通过交替学习两个组件以解耦依赖关系。在推理时，路由器实时分类输入，动态选择最佳提示，从而实现跨数据集干扰的有效缓解，同时保持单基准学习的高效性。
   - 结果：在多个数据集上，EEVEE在异构数据流下显著提升了鲁棒性：在Qwen3-4B-Instruct上平均多基准分数提升10.38，在DeepSeek-V3.2上提升24.32，超过SOTA方法GEPA和ACE最高达37.2%和48.2%。同时保持了单基准学习能力，证明了框架在真实世界部署中的有效性。

2. [TRACE: A Unified Rollout Budget Allocation Framework for Efficient Agentic Reinforcement Learning](https://arxiv.org/abs/2606.11119v1)
   - 分类：Agentic Reinforcement Learning / Budget Allocation
   - 研究价值：值得精读
   - 关联：解决了Agent RL中奖励对比度不足的关键问题，与EEVEE共同提升了Agent在复杂场景下的学习效率。
   - 痛点：在多轮Agent强化学习中，奖励对比度不足是主要瓶颈：过于简单或复杂的提示产生低方差反馈，且仅有结果奖励时，同一轨迹的所有步骤被赋予相同终端值，无法区分正确与错误决策。现有预算分配仅关注提示级信息，忽略了轨迹内部前缀级别的差异性，导致策略更新信号弱。TRACE通过建模树形展开结构，将预算分配粒度从提示根扩展到中间前缀，从根本上增强奖励对比。
   - 方法：TRACE将每个ReAct步骤（想法-动作-观察）视为语义不同的节点，从而形成树状展开。一个共享的可泛化预测器根据前缀历史估计条件成功概率，指导预算分配：优先将更多采样资源分配给那些最可能产生混合终端奖励的前缀。这种自适应树形结构丰富了反馈信号，放大了策略更新梯度，同时保持了采样预算固定。
   - 结果：在典型Agent基准上，TRACE在同等采样成本下提升了性能：Qwen3-14B在多跳QA任务上平均准确率提升2.8个百分点。实验还显示了在效率上的优势，证明树形预算分配比平面分配更有效，且能自然扩展到长尾场景。

3. [A History-Aware Visually Grounded Critic for Computer Use Agents](https://arxiv.org/abs/2606.11078v1)
   - 分类：Computer Use Agents / GUI Agent Critic
   - 研究价值：值得精读
   - 关联：为GUI Agent提供历史感知与视觉验证的批评机制，与现有的短视评论家相比有质的飞跃，可视为Agent自我纠正的关键组件。
   - 痛点：现有计算机使用Agent的评论家存在两个关键缺陷：一是短视，只关注当前动作而忽略更早的历史决策，导致重复错误或策略偏移；二是缺乏视觉根据，无法检测到错误点击等视觉级失败，例如点击错误UI元素。这些局限使得评论家难以在长时域GUI任务中提供有效纠正，限制了Agent的自主可靠性。HiViG通过历史记忆和视觉验证结合，从根本上解决了这些问题。
