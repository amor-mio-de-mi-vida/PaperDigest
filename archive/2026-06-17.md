1. [Unified Multimodal Autoregressive Modeling with Shared Context-Visual Tokenizer is Key to Unification](https://arxiv.org/abs/2606.18249v1)
   - 分类：Multimodal Understanding and Generation / Autoregressive Model
   - 研究价值：值得精读
   - 关联：该论文解决了多模态统一建模中分词器分裂问题，与DALL-E、Gemini等大型多模态模型直接相关。
   - 痛点：当前多模态统一模型（如DALL-E、Gemini）通常为理解和生成分别设计视觉分词器，导致表示空间分裂，需要额外转换模块并限制端到端优化。这种设计迫使模型在学习过程中权衡理解与生成性能，难以同时达到最优，成为真正统一多模态建模的核心瓶颈。
   - 方法：提出UniAR，使用单个共享离散视觉分词器桥接理解和生成。通过多级特征融合（结合ResNet和ViT特征）和无查找比特量化方案构建统一代码本，实现高语义和细节保留。基于此，统一自回归模型采用平行位预测（Parallel Bitwise Prediction）机制，将空间分组的视觉码（如32组）同时预测，将序列长度缩短至数百，大幅加速生成。最后，扩散解码器从离散码生成高保真图像。
   - 结果：在MS-COCO图像生成上取得SOTA FID；在InstructPix2Pix等编辑任务上表现最优；同时在VQAv2、COCO Captioning等多模态理解基准上保持竞争力。验证了共享分词器是统一多模态建模的关键要素，为构建通用多模态大模型提供了实用、高效的框架。

2. [Looped World Models](https://arxiv.org/abs/2606.18208v1)
   - 分类：World Model / Efficient Architecture
   - 研究价值：值得精读
   - 关联：该论文提供了世界模型的可扩展架构创新，与Dreamer、Transformer-based world models形成对比，提出深度循环作为新缩放轴。
   - 痛点：现有世界模型在长时序保真模拟和计算效率之间存在矛盾：深层模型能提高保真度但参数多、易积累复合误差；浅层模型效率高但预测质量有限。缺乏一种能根据预测难度自适应分配计算资源、同时参数高度共享的高效架构。
   - 方法：提出LoopWM，首个循环架构的世界模型。核心是使用参数共享的单个Transformer块对潜在状态进行多次迭代细化，每次迭代共享同一组参数。通过设定迭代次数或动态收敛准则，模型可自适应调整深度以匹配每一步预测的复杂度，实现计算资源的高效分配。
   - 结果：在多项世界模型基准上，LoopWM以不到传统方法1%的参数数量（效率提升100倍）达到相似或更好的预测保真度。自适应计算机制在简单步骤使用较少迭代，复杂步骤使用较多迭代，有效平衡了效率和精度。该工作开创了循环深度作为世界模型新缩放维度的方向。

3. [Future Dynamic 3D Reconstruction: A 3D World Model with Disentangled Ego-Motion](https://arxiv.org/abs/2606.18250v1)
   - 分类：Dynamic 3D Reconstruction / World Model
   - 研究价值：值得精读
   - 关联：与基于视频的world model不同，该工作将3D一致性引入未来预测，适用于自动驾驶等具身智能场景。
   - 痛点：现有生成式世界模型通常在2D图像空间进行未来预测，缺乏显式的3D几何结构，导致在长时序模拟中出现物体变形、消失等物理不一致问题，无法满足需要精确3D场景理解的具身决策任务。
   - 方法：提出FR3D，一个预测未来动态3D重建的世界模型。核心是显式解耦场景的3D演化与智能体自身运动：将智能体的自运动（ego-motion）作为潜在动作变量，与场景变化分离建模，解决自运动与世界运动混合导致的模糊性。同时，采用教师-学生蒸馏策略，利用预训练基础模型（如CLIP、DINO）的空间常识，实现零样本泛化到未见场景。
   - 结果：在多个室外动态场景数据集上，FR3D能从单目视频预测2秒后的3D场景，保持几何一致性和细节，显著优于基于2D视频预测的基线方法。消融实验证明解耦和蒸馏策略是关键。该工作为具身智能中的长期环境预测提供了可靠的3D世界模型。

**主要痛点**
- 多模态统一模型因分词器分裂难以实现真正统一；
- 世界模型在长时序模拟中参数效率低且易出现复合误差和几何不一致；
- 扩散策略因离散化伪影导致时序漂移，影响长周期部署鲁棒性；
- 多人-物体交互数据噪声大，存在接触、运动、手指细节伪影；
- 3D人脸编辑在变老任务中视图不一致导致过平滑；

**方法趋势**
- 共享视觉分词器与平行位预测（UniAR）；
- 循环架构与参数共享Transformer块实现自适应计算深度（LoopWM）；
- 解耦自运动与场景运动，结合蒸馏利用基础模型常识（FR3D）；
- 后缀条件自蒸馏与步骤级监督适配扩散LLM（d-OPSD）；
- Kolmogorov方程将扩散策略提升到Cameron-Martin空间，谱加权及PDE残差诊断；

**结果信号**
- UniAR在图像生成/编辑上达到SOTA，多模态理解保持竞争力；
- LoopWM参数效率提升100倍，自适应深度匹配复杂度；
- FR3D实现2秒后几何一致的动态3D重建；
- d-OPSD用10%优化步骤超越RLVR和SFT；
- Kolmogorov方法在PushT上提升17%奖励，减少67.6%漂移；

   - 分类：Language Model / Efficient Architecture
   - 研究价值：值得精读
   - 关联：提出非均匀宽度分配，显著提升参数与计算效率，直接应用于高效训练与推理。
   - 痛点：传统Transformer各层宽度固定，参数与计算预算均匀分配，忽略了不同层在计算中的不同角色，导致资源浪费且难以权衡模型容量与效率。
   - 方法：提出> <former架构，维持较宽的首尾层，通过参数无关的残差缩放机制收窄中间层，形成bottleneck结构，在不增加参数的前提下实现非均匀容量分配。
   - 结果：在200M到2B参数（dense）和3B参数（MoE）的语言模型上，一致优于参数匹配的均匀基线。在拟合损失匹配的缩放曲线下，减少22% FLOPs，KV缓存内存和I/O成本降低15%，证明非均匀宽度是实现资源最优缩放的有效路径。

2. [Ternary Mamba: Grouped Quantization-Aware Training of W1.58A16 State Space Models](https://arxiv.org/abs/2606.18114v1)
   - 分类：Model Quantization / State Space Model
   - 研究价值：值得精读
   - 关联：基于预训练检查点进行三元量化训练，大幅降低训练数据和时间成本，适用于边缘部署的高效推理。

   - 方法：提出分组量化感知训练（QAT）结合知识蒸馏，从冻结的FP16教师模型蒸馏至三元学生，仅需102M tokens（4 GPU小时）即可压缩3.61倍。引入零比率坍缩问题分析及解决方法。
   - 结果：Mamba-2 1.3B模型压缩至744MB（3.61×），在7任务平均零样本准确率48.1%，接近Bi-Mamba的48.4%（±0.9pp CI），证明基于预训练检查点的QAT是实现三元SSM的高效途径。
