### Diffusion / 生成式AI

**今日推荐精读 Top 3**

1. [Diffusion-Proof: Recipe for Formal Theorem Proving Beyond Auto-Regressive Generation](https://arxiv.org/abs/2606.19315v1)
   - 分类：Formal Reasoning / Diffusion LLM
   - 研究价值：值得精读
   - 关联：将扩散模型引入形式定理证明，拓展了生成式AI在逻辑推理领域的应用边界，对构建可解释AI系统有重要启示
   - 痛点：自回归语言模型在形式定理证明中面临长程依赖和错误累积问题，生成证明序列时容易丢失上下文一致性；扩散语言模型通过块级去噪生成具有天然的双向信息处理优势，但尚未被应用于形式数学推理领域，缺乏针对该场景的训练和推理方法。现有工作多关注自然语言，忽略了形式证明对严格逻辑和局部修正的特殊需求。
   - 方法：本文提出Diffusion-Proof框架，首次将扩散LLM应用于形式定理证明。训练两个模型：dLLM-Prover-7B用于整段证明生成，采用块级扩散训练；dLLM-Corrector-7B利用扩散模型的填充能力进行局部证明修正，通过双向信息（左右上下文）提高正确性。同时设计推理策略适配定理证明的逐步验证特点。
   - 结果：在MiniF2F-Test上取得6.14%的绝对提升，在ProofNet-Test上提升1.61%，显著优于同等规模的AR基线。尤为重要的是，成功解决了一个DeepSeek-Prover-V2-7B（更强模型）未能解决的IMO问题，证明了扩散模型在长程推理和错误恢复方面的独特优势。这表明扩散架构有望成为形式推理的新范式。

2. [DreamReasoner-8B: Block-Size Curriculum Learning for Diffusion Reasoning Models](https://arxiv.org/abs/2606.19257v1)
   - 分类：Diffusion Language Model / Reasoning
   - 研究价值：值得精读
   - 关联：解决了扩散语言模型在长链推理中的粒度问题，推动了扩散模型向推理密集型任务的延伸，与生成式AI的高效推理需求高度契合
   - 痛点：块扩散语言模型在长链推理任务中存在训练与推理块大小的粒度不匹配问题：使用大块训练会导致推理性能严重下降，而小块训练虽然有效但推理速度慢。这限制了扩散模型在需要高效推理的复杂数学和代码任务上的应用。现有研究缺乏系统性的解决方案。具体来说，大块训练使模型难以保持因果依赖和逻辑连贯性，导致推理能力退化。
   - 方法：本文提出块大小课程学习策略，训练时从细粒度小块（如2个token）开始，逐步过渡到粗粒度大块（如32个token），使模型学会适应不同块大小下的生成；同时发布DreamReasoner-8B开源模型，采用8B参数规模的扩散架构。推理时可灵活选择块大小，兼顾速度和质量。
   - 结果：在数学推理基准（如GSM8K）和代码生成基准（如HumanEval）上，DreamReasoner-8B达到与Qwen3-8B（自回归模型）相当的性能，同时推理速度更快（得益于并行去噪）。消融实验验证了课程学习的有效性，大块训练性能从极低提升到可竞争水平。开源模型促进了扩散推理研究的可复现性。

3. [NeuMesh++: Towards Versatile and Efficient Volumetric Editing with Disentangled Neural Mesh-based Implicit Field](https://arxiv.org/abs/2606.19316v1)
   - 分类：3D Scene Editing / NeRF
   - 研究价值：值得精读
   - 关联：基于网格的隐式场编辑为3D内容生成提供灵活高效的工具，是生成式AI在3D交互编辑方向的重要进展
   - 痛点：现有基于神经隐式场的3D编辑方法功能受限，大多仅支持刚体变换或类别特定语义编辑，无法实现灵活的局部几何变形、纹理置换和语义引导编辑；同时编辑过程往往需要重新训练网络或依赖繁琐的手动标注，缺乏高效的交互式编辑手段。具体来说，用户难以对NeRF重建的场景进行直观的几何结构调整（如物体形状变化）、精细纹理修改（如颜色填充和纹理迁移），以及基于语义的区域选择编辑；这些痛点严重制约了神经渲染在数字内容创作等实际应用中的可用性和效率。因此亟需一种能解耦场景不同属性并支持多种编辑操作的高效表示框架。
   - 方法：本文提出基于网格的神经隐式场NeuMesh++，将场景的几何、纹理和语义信息解耦编码到网格顶点上的隐向量中；通过网格拓扑引导实现几何编辑，通过纹理码替换和可学习颜色修改实现纹理编辑，通过语义码辅助区域选择实现语义编辑；并引入局部空间参数化增强渲染质量，空间感知优化策略实现精确纹理修改。此外，语义辅助区域选择减少了手动标注工作量。整体框架在训练时保持端到端优化，编辑时无需重新训练网络。
   - 结果：在真实和合成数据集上的大量实验表明，NeuMesh++在渲染质量和编辑能力上均优于现有方法。在NeRF-Synthetic和真实场景数据集上，PSNR达到28.5dB，比现有最佳编辑方法高1.2dB；纹理编辑成功率和语义编辑准确率均有显著提升；编辑操作平均耗时仅0.5秒，支持实时交互；且编辑后的场景可保证多视角一致性。用户研究也证明编辑结果更符合预期。

**主要痛点**
- 扩散语言模型在长链推理中面临训练与推理块大小粒度不匹配问题
- 自回归模型在形式定理证明中存在错误累积和长程依赖缺陷
- 现有3D神经渲染编辑方法功能单一，缺乏灵活的体积编辑能力

**方法趋势**
- 块大小课程学习策略
- 首次将扩散LLM应用于形式定理证明
- 基于网格的解耦隐式场表示

**结果信号**
- DreamReasoner-8B达到与自回归模型相当的性能
- Diffusion-Proof在定理证明基准上显著提升并解决IMO难题
- NeuMesh++实现多种高效编辑操作

1. [Zero-Shot Long-Horizon Dexterous Manipulation via Multi-View 3D-Grounded VLM Reasoning](https://arxiv.org/abs/2606.19340v1)
   - 分类：Dexterous Manipulation / VLM Reasoning
   - 研究价值：值得精读
   - 关联：零样本灵巧操作框架，结合多视图3D和VLM推理，直接应用于具身智能场景
   - 痛点：现有灵巧操作方法依赖任务特定训练或预定义技能库，无法零样本泛化到未见过的物体和工具使用任务，且长时序操作中的3D接地和闭环重规划困难。
   - 方法：提出零样本框架，使用VLM从多视图RGB图像生成参考帧任务接地和2D关键点，通过三角测量和参考视图射线投票融合为3D关键点；对工具使用检索原子动作并对齐6D轨迹，对灵巧抓取扩展为任务条件抓取区域并生成抓取-运动对。
   - 结果：在真实世界实验中，相比单视图RGB-D接地和微调VLA基线，3D接地精度和执行可靠性显著提升，支持对未见物体和工具使用任务的零样本长时序操作。

2. [Do as I Do: Dexterous Manipulation Data from Everyday Human Videos](https://arxiv.org/abs/2606.19333v1)
   - 分类：Dexterous Manipulation / Data Generation
   - 研究价值：值得精读
   - 关联：从众包人类视频中提取灵巧操作数据，解决机器人学习数据稀缺问题
   - 痛点：灵巧多指手操作数据难以规模化采集，现有人类视频到机器人数据迁移受限于手-物交互估计不准确和人-机具身差异。
   - 方法：提出DO AS I DO算法，从单目RGB人类视频中重建手-物交互（包括自中心和自中心视角），然后通过重定向将估计的交互序列转化为多指灵巧手可执行的机器人动作序列，生成完整操作数据。
   - 结果：在多个含真值数据集和在线视频片段上，手-物交互估计和灵巧操作轨迹提取效果优于先前方法，并提出了从业者收集人类数据的效果手册。

3. [STARE: Surprisal-Guided Token-Level Advantage Reweighting for Policy Entropy Stability](https://arxiv.org/abs/2606.19236v1)
   - 分类：RLHF / Policy Entropy
   - 研究价值：值得精读
   - 关联：解决GRPO训练中策略熵崩溃问题，直接提升RL在推理任务中的稳定性和性能
   - 痛点：GRPO等RL算法在LLM后训练中常导致策略熵崩溃，现有方法缺乏对令牌级信用分配和熵动态的深入分析，导致训练不稳定和探索不足。
   - 方法：通过梯度分析识别令牌熵变化与优势-惊奇四象限结构，提出STARE：利用批次内惊奇分位数识别熵关键令牌子集，选择性重加权其有效优势，并引入目标熵闭环门控稳定调节。
   - 结果：在1.5B至32B模型及多种推理任务上，STARE在数千步训练中维持策略熵稳定，在AIME24/25上比DAPO等基线平均提升4%-8%，且反射令牌和响应长度同步增长。

**主要痛点**
- 灵巧操作数据匮乏且零样本泛化困难
- 强化学习训练中策略熵崩溃导致探索不足
- VLA模型微调后常识知识保留不明
- 偏好强化学习样本效率低，被动探索
- 3D场景理解依赖复杂几何编码器，计算成本高

**方法趋势**
- 多视图3D接地VLM推理实现零样本操作
- 从人类视频重建并重定向灵巧操作数据
- 令牌级惊奇引导优势重加权稳定策略熵
- 不确定性平衡偏好规划主动探索
- 全景重投影实现高效3D场景理解

**结果信号**

1. [Zero-Shot Long-Horizon Dexterous Manipulation via Multi-View 3D-Grounded VLM Reasoning](https://arxiv.org/abs/2606.19340v1)
   - 分类：Dexterous Manipulation / VLM Grounding

   - 关联：提出零样本灵巧操作框架，弥合VLM与机器人执行之间的鸿沟，通过多视图3D融合提升泛化能力。

2. [Native Active Perception as Reasoning for Omni-Modal Understanding](https://arxiv.org/abs/2606.19341v1)
   - 分类：Video Understanding / Active Perception / Omni-Modal Agent
   - 研究价值：值得精读
   - 关联：将视频理解建模为POMDP迭代过程，实现主动感知，首次在开源模型上超越10倍参数量的闭源模型。

   - 方法：提出OmniAgent，基于POMDP的迭代观察-思考-行动循环，选择性提取音视频线索并存入持久化文本记忆；引入智能体监督微调（利用best-of-N轨迹合成）和基于TAURA（回合感知自适应不确定性重整优势）的强化学习，驱动模型在关键推理回合产生有效探索。

3. [Rethinking Reward Supervision: Rubric-Conditioned Self-Distillation](https://arxiv.org/abs/2606.19327v1)
   - 分类：RL for LLM / Reasoning Model
   - 研究价值：值得精读
   - 关联：引入评分标准作为细粒度反馈，改进蒸馏和强化学习范式，在科学推理任务上超越GRPO。
   - 痛点：现有推理模型后训练依赖链式思维蒸馏（成本高且可能存在噪声）或标量奖励强化学习（无法区分回答中各部分优劣），导致信用分配不精确，模型难以定位具体改进点。
   - 方法：提出基于评分标准的自蒸馏框架：先学习生成任务特定评分标准，再以评分标准为条件训练教师模型，对学生自身的采样轨迹提供token级指导；评分标准明确强回答需满足的条件，实现比标量奖励更细粒度的信用分配。
   - 结果：在多个科学推理基准上平均超越GRPO 1.0个点、OPSD 0.9个点；评分标准有效将标准级标准转换为token级指导，显著提升推理过程质量。

**主要痛点**
- 机器人灵巧操作零样本泛化困难，3D定位精度不足
- 长视频理解计算成本随时长线性增长，缺乏主动感知机制
- LLM后训练中奖励监督粗粒度，信用分配不精确
- 多智能体决策中立场依赖难以处理
