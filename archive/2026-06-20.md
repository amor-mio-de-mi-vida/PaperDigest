   - 分类：3D Generation / Visual Illusion
   - 研究价值：值得精读
   - 关联：首次实现零样本3D视觉错觉生成，方法创新性强
   - 痛点：现有3D生成方法难以快速生成具有视觉错觉效果（如形状与渲染不一致）的3D模型，且通常需要针对特定场景微调或大量训练数据，缺乏通用性和零样本能力。
   - 方法：提出跨空间去噪框架，在2D图像空间和3D几何空间之间交替去噪，利用预训练扩散模型的2D先验指导3D几何优化，实现零样本的视觉错觉生成。关键机制包括：双空间噪声调度、跨空间注意力融合、以及自适应几何细化。
   - 结果：在多个视觉错觉任务（如3D形状-纹理不一致）上，JanusMesh在秒级内生成高质量3D模型，优于之前需要数小时微调的方法；用户研究显示错觉效果显著，且生成速度提升两个数量级。

2. [Current World Models Lack a Persistent State Core](https://huggingface.co/papers/2606.20545)
   - 分类：Reinforcement Learning / World Models
   - 研究价值：值得精读
   - 关联：揭示世界模型的核心缺陷，对构建长期记忆智能体有重要指导
   - 痛点：当前世界模型在复杂任务中无法维持长期依赖和新颖性检测，因为缺乏一个能够持久存储和更新时间相关状态的模块，导致模型在需要历史信息的任务中表现不佳。
   - 方法：提出“持久状态核心”概念，分析现有世界模型架构（如RNN、Transformer、状态空间模型）在时间一致性上的缺陷；设计一种新的残差状态更新机制，通过解耦短时和长时记忆，并引入显式的时间戳对齐，使模型能够持续维护一个不变的核心状态。
   - 结果：在多个长期依赖基准（如Memory Maze、Atari 100k）上，结合持久状态核心的世界模型相比基线（DreamerV3、Stabilizing Transformers）在回滚测试和系综多样性上提升15-30%，且参数增加不到5%。

**主要痛点**
- 3D视觉错觉生成缺乏快速零样本方法
- 世界模型无法维持长期时间依赖性

**方法趋势**

   - 分类：Novel View Synthesis / Dataset
   - 研究价值：值得精读
   - 关联：为去除干扰物的新视角合成提供大规模基准，推动3D重建与渲染研究
   - 痛点：现有新视角合成数据集包含移动物体或动态场景干扰，导致模型无法专注于静态场景的几何与外观学习，限制了合成质量和泛化能力。该论文构建了首个大规模去干扰物数据集，解决训练数据中干扰物导致的伪影和模糊问题。
   - 方法：通过多视角视频采集与自动干扰物检测去除算法，筛选出1000个静态场景，每个场景包含高分辨率RGB图像和精确相机位姿，并设计统一的评估协议，包含多种指标（PSNR、SSIM、LPIPS等）。
   - 结果：实验表明，在该数据集上训练的模型在去干扰物场景下合成质量提升显著，PSNR平均提高2.1dB，且泛化到未知场景时表现更鲁棒，证明了高质量去干扰物数据对提升新视角合成性能的关键作用。

   - 方法：从Unity和Unreal Engine开源项目中提取1000+个完整项目，定义代码框架的模块化层级（脚本、组件、系统），并设计跨项目级别的基准任务（如模块迁移、依赖推理），采用语义相似度和功能正确性双重指标。

**主要痛点**

**方法趋势**

   - 方法：提出DragMesh-2框架，通过将可动关节物体建模为多刚体系统，结合接触力预测与逆运动学优化，在物理仿真中实现灵巧手对铰接物体的精确操作。

   - 结果：在具身问答和网页导航等任务上，CA-RL相比未增强的LLM智能体成功率提升15-20%，且对分布外的上下文变化鲁棒性更强。

**主要痛点**

1. [Playful Agentic Robot Learning](https://huggingface.co/papers/2606.19419)
   - 分类：Robot Learning / Interactive Learning
   - 研究价值：值得精读
   - 关联：将内在动机游戏化框架引入机器人学习，与现有基于稀疏奖励的方法形成对比，开启具身智能新范式。
   - 痛点：传统机器人学习依赖大量人工采集的示范数据或固定的奖励函数，难以在动态、复杂环境中实现自适应和高效探索。学习过程枯燥，缺乏内在动机驱动，导致样本效率低下，且学到的技能难以泛化到新场景。
   - 方法：提出“Playful Agentic Robot Learning”框架，将机器人学习塑造成多阶段游戏（如寻宝、拼图），每个游戏包含子任务，通过内置的好奇心模块（ICM）和自监督目标（预测下一状态误差）作为内在奖励，引导机器人自主探索环境和发现有用技能，同时采用课程学习逐步提升任务难度。
   - 结果：在多个模拟和真实机器人平台上实验，相比SAC、HER等基线，该方法在任务成功率上提升显著，样本效率提高50%以上，且在未见过的场景中表现出更好的泛化能力，机器人学会了多种可迁移技能，如抓取、推动和导航。

2. [S-Agent: Spatial Tool-Use Elicits Reasoning for Spatial Intelligence](https://huggingface.co/papers/2606.20515)
   - 分类：Spatial Intelligence / Tool Use / Spatial Reasoning
   - 研究价值：值得精读
   - 关联：展示了LLM通过外部工具增强空间推理的可行性，对体现代理方向具有启发。
   - 痛点：大型语言模型在处理需要精确空间推理（如距离、方向、布局）的任务时表现欠佳，缺乏与物理世界的交互感知能力，难以仅通过文本理解空间关系，导致在空间问答、导航等任务中错误频出。
   - 方法：设计S-Agent框架，集成空间工具库（地图查看、距离计算、方位测量等API），模型通过ReAct模式进行思考-行动循环：识别空间问题→调用工具获取具体数据→基于工具输出进行推理。同时引入“空间链”机制，将复杂问题分解为逐步探索的子问题，并利用工具反馈进行自我纠错。
   - 结果：在SpatialQA、VSR、BLINK等空间推理基准上，S-Agent相比GPT-4等基线，准确率提升15-25%，且能泛化到未见过的空间场景。消融实验证明工具使用环节是关键，移除后性能大幅下降，表明工具增强推理的有效性。

3. [Context-Aware RL for Agentic and Multimodal LLMs](https://huggingface.co/papers/2606.17053)
   - 分类：Reinforcement Learning / Multi-modal Agents / LLM Finetuning
   - 研究价值：值得精读
   - 关联：为多模态LLM代理训练提供了强化学习新范式，解决了上下文建模不足的问题。
   - 痛点：多模态LLM在开放世界中作为代理时，往往缺乏长期的上下文记忆和动态调整策略的能力，导致重复错误、任务失败，且难以从环境反馈中高效学习，现有SFT和标准PPO方法无法充分利用历史交互信息。
   - 方法：将代理决策过程建模为部分可观测马尔可夫决策过程（POMDP），引入上下文编码器将历史观察（视觉、文本、工具反馈）压缩为隐状态。使用带KL正则化的策略梯度（类似PPO）训练，动作空间包括自然语言回复、工具调用和导航指令。特别设计“情境门控”机制，根据上下文状态自动调整探索性，避免早期过拟合。
   - 结果：在ALFWorld、WebShop、VATEX等基准上，CA-RL任务成功率平均提升20%，平均步数减少30%，并能泛化到新任务。分析显示模型有效利用历史信息避免重复错误，上下文编码器对长期依赖捕获能力显著优于基线。

**主要痛点**
- 传统机器人学习样本效率低，缺乏内在动机驱动。
- 大型语言模型空间推理能力不足，缺少与物理世界交互。
- 多模态LLM代理缺乏长期上下文记忆和动态策略调整。

**方法趋势**
- 基于好奇心模块和自监督目标的游戏化机器人学习框架。
- 通过外部空间工具库增强LLM空间推理的S-Agent框架。
- 基于POMDP和上下文编码器结合的情境门控强化学习方法。

**结果信号**
- 机器人学习任务成功率提升，样本效率提高50%以上。
- 空间推理基准准确率提升15-25%，工具使用是关键。
