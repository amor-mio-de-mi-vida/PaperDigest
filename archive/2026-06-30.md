1. [Training Vision-Language-Action Models with Dense Embodied Chain-of-Thought Supervision](https://arxiv.org/abs/2606.30552v1)
   - 分类：Vision-Language-Action / Embodied AI
   - 研究价值：值得精读
   - 关联：提出ECoT跨本体迁移方法，大规模数据集ProcCorpus-60M，开源模型
   - 痛点：跨本体迁移是VLA模型的核心挑战，因为不同机器人的低层级状态和动作空间完全不同，导致模型难以泛化。现有方法通常需要针对每个本体重新训练或大量微调，缺乏共享的高层认知表示。
   - 方法：提出ZR-0模型，采用双流架构：预训练VLM（System 2）在训练时生成密集的Embodied Chain-of-Thought（ECoT）推理，涵盖场景感知、物体识别、任务规划等；Diffusion Transformer动作专家（System 1）通过流匹配生成连续动作块。两者通过交叉注意力耦合，推理时可完全跳过ECoT生成，无损性能。
   - 结果：在单臂（LIBERO）、双臂（RoboTwin 2.0）和人形（RoboCasa GR-1 Tabletop）三个模拟平台以及真实xArm平台上均取得领先性能，ECoT覆盖96.8%帧，验证了跨本体迁移的有效性。

2. [LeVo 2: Stable and Melodious Song Generation via Hierarchical Representation Modeling and Progressive Post-Training](https://arxiv.org/abs/2606.30642v1)
   - 分类：Song Generation / Hybrid LLM-Diffusion
   - 研究价值：值得精读
   - 关联：混合LLM-Diffusion框架，层次化建模，渐进式后训练，开源基线对比
   - 痛点：全长度歌曲生成需兼顾连贯性、声学细节和歌词跟随，现有语言模型方法在混合token建模与双轨预测之间存在结构性权衡：混合token保持声乐-伴奏协调但模糊轨道细节，双轨预测改善声学但序列长且全局规划弱。
   - 方法：提出LeVo 2，层次化建模框架：LeLM先预测混合token进行语义规划，再并行预测声乐和伴奏token进行轨道细化，扩散Music Codec重建波形。核心贡献是美学引导的训练流程：预训练时用自动美学评估分配音乐性条件，后训练依次使用SFT、大规模离线DPO和闭环半在线DPO，分阶段提升质量、可控性和音乐性。
   - 结果：专家听感和客观评估显示，LeVo 2在六个主观维度上超越开源基线，在多个听感指标上接近领先商业系统。消融验证了训练策略、美学引导、缩放和层次架构的有效性。

3. [StereoGS: Sparse-View 3D Gaussian Splatting via Stereo Priors](https://arxiv.org/abs/2606.30545v1)

   - 结果：在LLFF、DTU、Mip-NeRF360和Blender数据集上，稀疏视图设置下达到SOTA性能，且不增加推理开销。

1. [One-Step Gradient Delay is Not a Barrier for Large-Scale Asynchronous Pipeline Parallel LLM Pretraining](https://arxiv.org/abs/2606.30634v1)
   - 分类：Pipeline Parallelism / Optimizer Robustness
   - 研究价值：值得精读
   - 关联：解决异步流水线并行中梯度陈旧性导致的训练不稳定问题，直接提升大规模LLM预训练效率。
   - 痛点：异步流水线并行虽然消除了同步气泡，但梯度陈旧性问题被认为会导致训练不稳定，阻碍其大规模应用。此前普遍认为陈旧性是固有缺陷，缺乏系统的优化器鲁棒性分析。
   - 方法：首次系统实证分析不同优化器（AdamW vs Muon）对一步梯度延迟的鲁棒性，发现Muon具有天然鲁棒性。提出优化器无关的Error Feedback修正方法，从理论上证明Muon在有无修正下的收敛性，并在10B参数模型上验证。
   - 结果：AdamW在一步延迟下严重退化，而Muon保持鲁棒；Error Feedback修正进一步弥合与同步训练的性能差距。10B模型实验证实异步流水线并行可达到同步训练的准确率，同时消除气泡，大幅提升吞吐量。

2. [Scaling the Horizon, Not the Parameters: Reaching Trillion-Parameter Performance with a 35B Agent](https://arxiv.org/abs/2606.30616v1)
   - 分类：Agentic Training / Mixture of Experts
   - 研究价值：值得精读
   - 关联：提出通过扩展智能体轨迹长度而非模型参数来提升性能，为高效训练提供新范式。
   - 痛点：当前LLM训练主要依赖扩展模型参数规模，但成本极高。如何用较小模型（35B）达到万亿参数模型的性能？关键在于提升“智能体水平”（agent horizon），但缺乏有效的长轨迹训练框架和异构能力融合方法。
   - 方法：构建长轨迹知识-动作基础设施（平均45K tokens/轨迹），采用三阶段训练：全域SFT对齐基础模型、领域级教师模型捕获专长、多教师领域路由在线蒸馏（含显著词汇对齐）。通过多教师蒸馏将6个异构领域能力统一到单个35B MoE学生模型。
   - 结果：35B Agents-A1在SEAL-0、IFBench、HiPhO等多个长时任务基准上超越Kimi-K2.6、DeepSeek-V4-pro等万亿参数模型，在长时任务上达到或匹配1T模型性能。证明扩展轨迹长度是参数扩展的有效替代。

3. [Morphing into Hybrid Attention Models](https://arxiv.org/abs/2606.30562v1)
   - 分类：Hybrid Attention / Long-Context Efficiency
   - 研究价值：值得精读
   - 关联：提出自动化层选择方法将Transformer转换为高效混合注意力模型，直接提升长上下文推理效率。
   - 痛点：混合注意力模型（部分全注意力+部分线性注意力）可提升长上下文效率，但关键在于选择哪些层保留全注意力。现有启发式或逐层评分方法忽略层间相互依赖，导致次优配置，且选择成本高。
   - 方法：将混合层选择形式化为预算约束的子集优化问题，提出FlashMorph：先构建可变形模型（每层附加线性注意力分支），冻结权重后在合成长上下文检索数据上联合优化逐层门控（含线性化正则化），再将门控离散化得到混合架构，最后进行logits蒸馏和长上下文微调。
   - 结果：FlashMorph发现更优混合配置，在保持强长上下文召回和通用基准性能的同时，大幅降低层选择成本。实验证明其优于现有层选择方法，实现效率与效果的最优平衡。

**主要痛点**
- 异步流水线并行中梯度陈旧性导致的不稳定问题。
- 扩展模型参数到万亿级别的高成本。
- 混合注意力模型中全注意力层选择缺乏系统方法。
- on-policy蒸馏中的特权幻觉和冲突信号。
- CNN模型部署时缺乏无需数据的动态压缩方法。

**方法趋势**
- Muon优化器鲁棒性与Error Feedback修正。
- 扩展智能体轨迹长度和多教师蒸馏。
- FlashMorph：基于可变形模型和门控优化的层选择。
- DOPD：优势感知的双重蒸馏动态路由。
- HASTE：基于局部敏感哈希的通道合并。

**结果信号**
- 异步流水线并行达到同步训练精度，消除气泡。
- 35B模型在长时任务上超越万亿参数模型。

- DOPD在LLM和VLM上稳定超越基线。

1. [Training Vision-Language-Action Models with Dense Embodied Chain-of-Thought Supervision](https://arxiv.org/abs/2606.30552v1)

   - 研究价值：值得精读
   - 关联：核心具身智能方法，解决跨embodiment迁移问题
   - 痛点：跨实体（cross-embodiment）的视觉-语言-动作（VLA）模型迁移是具身智能的核心挑战。不同机器人平台的低层状态和动作空间差异巨大，导致直接迁移困难。然而，高层认知过程（场景感知、物体识别、任务规划、子任务分解等）本质上是跨实体共享的，现有方法未能显式利用这一共享表示来对齐不同实体的特征空间。本文旨在通过稠密的Embodied Chain-of-Thought监督来桥接这一表示鸿沟。
   - 方法：ZR-0采用双流架构：一个预训练的视觉-语言模型（VLM）作为System 2，在训练时为每个帧生成结构化的Embodied Chain-of-Thought (ECoT)推理（包括场景描述、物体定位、任务规划等）；一个基于扩散变换器的动作专家（System 1）通过流匹配生成连续动作块。两个流通过交叉注意力耦合，并通过注意力掩码确保推理时ECoT生成可完全跳过且无性能损失。模型在ProcCorpus-60M大规模数据集上预训练，该数据集包含约6000万帧、超过40万条轨迹，其中96.8%的帧带有ECoT标注。
   - 结果：ZR-0在三个仿真基准（单臂LIBERO、双臂RoboTwin 2.0、人形RoboCasa GR-1）以及xArm真实机器人平台上均取得强劲表现，证明了其跨实体泛化能力。通过显式建模共享认知过程并耦合动作生成，ZR-0在多个任务上超越了此前方法。代码和模型已开源。

   - 关联：人形机器人操作，合成数据生成，Sim-to-Real
   - 痛点：基于感知的人形机器人全身运动操作（loco-manipulation）需要连接自我中心观察、语言指令和机器人兼容的运动轨迹。然而，目前没有数据源能同时提供同步的自我中心图像、语言命令和运动轨迹的大规模数据。这成为训练此类策略的瓶颈。

   - 结果：在物理Unitree G1人形机器人上评估导航和单物体运输任务，证明合成交互在重建场景中可以为基于sim-to-real的感知人形机器人物操作提供有效监督。该方法实现了从仿真到真实的有效迁移。

2. [Forensic Trajectory Signatures for Agent Memory Poisoning Detection](https://arxiv.org/abs/2606.30566v1)
   - 分类：Agent Security / Memory Poisoning
   - 研究价值：值得精读
   - 关联：发现了持久内存中毒攻击下LLM代理的行为不变性，为检测代理内存攻击提供了高效且可泛化的方法。
   - 痛点：LLM代理越来越多地使用持久内存，但内存中毒攻击难以检测。现有方法需要攻击轨迹或复杂模型，且缺乏可解释性和跨模型泛化能力。
   - 方法：发现关键行为不变性：成功攻击必须先在可观察内存工具中调用memory_recall_fact然后调用email_send_email，该转换在非攻击会话中极少出现。基于此简单规则达到AUC 0.9563。进一步使用19个轨迹特征的随机森林分类器达到AUC 0.9904，且移除一半特征后AUC不变，表明攻击印迹是多通道的。方法可零训练泛化到GPT-4.1等前沿模型。
   - 结果：在9个模型（7B-120B）上，跨模型留出验证中6/9 AUC=1.000，所有异常有机制解释。严格前缀变体AUC=0.934，支持实时检测。还可区分内存中毒攻击和提示注入攻击。

3. [Scaling the Horizon, Not the Parameters: Reaching Trillion-Parameter Performance with a 35B Agent](https://arxiv.org/abs/2606.30616v1)
   - 分类：Agentic Models / MoE / Scaling
   - 研究价值：值得精读
   - 关联：提出通过扩展代理时间线而非模型参数规模达到万亿参数性能的新范式，并开源了训练流程和基础设施。
   - 痛点：当前扩展定律聚焦模型参数，导致计算成本极高。探索通过扩展代理交互时间线（horizon）来提升性能的路径，但缺乏长时域轨迹数据和训练方法。
   - 方法：构建长时域知识-动作基础设施，连接外部知识、动作、观察和验证器结果，产生平均45K token的代理轨迹。三阶段训练：全领域SFT对齐基础模型；训练领域级教师模型捕获专业能力；多教师领域路由在线蒸馏，结合显著词汇对齐来改进知识迁移效率，统一6个异构领域到一个可部署学生模型Agents-A1（35B MoE）。
