1. [SynCity 3000: Bootstrapping Scene-Scale 3D Diffusion](https://arxiv.org/abs/2607.05392v1)
   - 分类：Scene-Scale 3D Generation / Diffusion
   - 研究价值：值得精读
   - 关联：提出场景级3D生成框架，解决现有方法在规模和一致性上的局限。
   - 痛点：现有3D生成方法局限于单个物体或小场景，无法生成全局一致且细粒度可控的大规模3D场景。缺乏用于训练场景级模型的数据，且直接扩展图像到3D模型到场景尺度面临计算和一致性问题。当前方法要么输出碎片化，要么依赖复杂管线，难以实现任意大小和复杂度的场景生成。
   - 方法：通过微调单图生成3D模型为卷积算子，使其可滑动地应用于整个场景的dimetric图像，从而生成任意尺寸的3D场景。为克服训练数据稀缺，提出合成数据引擎自主生成场景级训练数据，驱动模型适应场景级分布。整体框架结合用户提示驱动的全景图像生成和卷积式3D重建。
   - 结果：在多种提示和布局下，SynCity 3000生成的大尺度3D场景在全局一致性、细节丰富度和布局可控性上显著优于先前方法，例如场景结构连贯无断裂，物体布局符合语义，且支持用户指定的细粒度控制，为场景级3D生成设立了新基准。

2. [Search Beyond What Can Be Taught: Evolving the Knowledge Boundary in Agentic Visual Generation](https://arxiv.org/abs/2607.05382v1)
   - 分类：Visual Generation / Knowledge Boundary / Agentic
   - 研究价值：值得精读
   - 关联：揭示视觉生成模型的世界知识瓶颈，并提出教-搜协同训练框架。
   - 痛点：视觉生成模型在未见过的长尾实体（新角色、新事件等）上会自信地虚构内容，源于训练语料固定而视觉世界开放。现有基准无法捕捉这种缺陷，导致模型实际表现远低于预期。仅靠检索增强的简单方案会引入噪声，破坏模型已掌握的知识，因此需要精确划分生成器的知识边界。
   - 方法：构建SearchGen-20K基准（20839个提示，12类失败模式，22个领域）和1M条预执行搜索语料。核心创新是教-搜协同训练框架：先通过多轮搜索与微调让模型学会内部化可学习的知识，再依赖外部搜索补充边界外知识。利用生成器特定的、动态演化的知识边界，实现迭代自改进。
   - 结果：在SearchGen-Bench上，前沿开放生成器得分仅21-28/100（远低于现有基准的60+），证明现有评估严重低估问题。所提协同训练框架即使最小版本也带来单调提升，最终模型在长尾知识查询上显著提升准确性，同时保留原有生成能力，为工具增强的视觉生成奠定基础。

3. [Multiplayer Interactive World Models with Representation Autoencoders](https://arxiv.org/abs/2607.05352v1)
   - 分类：World Models / Multi-Agent / Diffusion
   - 研究价值：值得精读
   - 关联：首个多人世界模型，在复杂物理环境中实现长时间稳定推理。
   - 痛点：现有世界模型多针对单智能体场景，无法处理多智能体高度动态环境下的复杂物理交互和动作组合。当多个智能体同时行动时，模型需要正确归因场景变化并保持长期一致性，现有方法在长时域推理中容易崩溃或产生物理不一致。

   - 结果：在单个Nvidia B200 GPU上实时生成20fps的四玩家比赛，rollout稳定性远超训练时长：分布质量在5分钟内保持稳定，实际观察中rollout可持续数小时不崩溃。系统分析了模型和数据规模的影响，发现涌现出策略性行为，同时存在一些失败模式。提供了完整数据集、代码和实时演示。

**主要痛点**

### 高效训练与推理

1. [Weak-to-Strong Generalization via Direct On-Policy Distillation](https://arxiv.org/abs/2607.05394v1)
   - 分类：Efficient RL Training / Knowledge Distillation
   - 研究价值：值得精读
   - 关联：高效训练：弱到强泛化
   - 痛点：大型语言模型在可验证奖励强化学习（RLVR）框架中，每训练一代新强模型都需要从零开始生成大量rollout，计算和存储成本极度高昂；尝试将弱模型RL训练后的策略直接蒸馏给强模型，但弱模型最终策略混合了RL增益和小模型局限性，导致迁移效果差，弱模型训练成果被浪费。
   - 方法：Direct-OPD提出比较弱模型RL训练前后检查点的对数概率比（log-ratio），将其作为隐式密集奖励信号，直接施加在强模型的on-policy状态下进行训练；该方法无需训练显式奖励模型，也无需在目标模型上运行稀疏奖励RL，仅通过复用弱模型的RL监督信号即可实现高效的知识迁移。
   - 结果：在AIME 2024基准上，Qwen3-1.7B准确率从48.3%提升至62.4%，训练仅耗时4小时（8×A100 GPU）；相比直接蒸馏和步骤匹配的RL方法，Direct-OPD表现更优，并且支持多个策略偏移的顺序组合，验证了RL结果可作为隐式奖励跨模型规模高效复用。

2. [TREK: Distill to Explore, Reinforce to Refine](https://arxiv.org/abs/2607.05339v1)
   - 分类：Efficient RL Training / Distillation
   - 研究价值：值得精读
   - 关联：高效训练：探索扩展与微调
   - 痛点：在强化学习微调中，GRPO等on-policy方法当当前策略无法采样到正确推理路径时，探索会停滞，无法利用支持集之外的优质解，导致模型在困难提示上长期表现不佳；而传统蒸馏或模仿学习只能压缩现有知识，无法扩展探索空间。
   - 方法：TREK通过两阶段策略：首先识别学生模型通过率极低的提示，使用教师模型（外部或自上下文）生成验证过的候选解，并按学生似然排序保留top-k；然后进行短时前向KL蒸馏，将这些解拉入学生模型的支持集；最后返回标准on-policy GRPO进行微调，从而高效扩展探索空间。
   - 结果：在数学推理上，TREK使用DeepSeek-V4教师使Qwen3-8B在AIME 2025上从36.9提升至40.3，AIME 2024从47.9提升至51.1（avg@16）；自上下文变体无需外部教师分别达到38.5和49.6。在ALFWorld（75.8→82.8）和ScienceWorld（12.5→26.7）智能体任务上同样显著提升，且早期训练效率更高。

3. [Advances in Neural Controlled Differential Equations](https://arxiv.org/abs/2607.05280v1)
   - 分类：Efficient Time Series Modeling / Neural ODEs
   - 研究价值：值得精读
   - 关联：高效训练：NCDE加速
   - 痛点：神经控制微分方程（NCDE）因其非线性向量场和顺序求解过程，训练计算成本极高，前向传播耗时，难以扩展到大规模时间序列数据和实际应用；传统NCDE在训练过程中需要密集的数值积分，导致每步计算量巨大。
   - 方法：本论文通过三条路径改进：Log-NCDE利用Log-ODE方法近似NCDE解，在训练中加快速度；Linear NCDE采用线性向量场替代非线性，使得方程存在闭式解，从而实现并行计算；Structured Linear NCDE进一步使用结构化线性场，在保持理论表达性的前提下提升计算效率。
   - 结果：在多种时间序列基准上，三种方法将NCDE每步训练时间降低最多三个数量级，同时达到或超越现有最优性能；Linear NCDE在并行计算下显著加速，而Structured版本在效率和性能间取得最佳平衡。

**主要痛点**
- 大型语言模型RLVR训练成本高，弱模型成果难以高效复用。
- 探索空间受限，on-policy方法在困难提示上停滞。
- NCDE训练计算昂贵，顺序求解限制可扩展性。
- 集成方法调参成本高，静态批处理策略无法适应异质负载。
- 视频流式推理分辨率与延迟矛盾。

**方法趋势**
- Direct-OPD：利用弱模型RL前后log-ratio作为隐式奖励直接迁移。
- TREK：前向KL蒸馏扩展探索支持结合GRPO微调。
- Log-NCDE/Linear NCDE：近似或闭式解加速NCDE训练。
- TabPack：并行采样多超参数MLP并动态选择集成。
- GSS：共享采样未来轨迹避免树搜索指数分支。

**结果信号**
- Direct-OPD：Qwen3-1.7B在AIME 2024上提升至62.4%。
- TREK：Qwen3-8B在AIME 2025提升至40.3，ALFWorld成功率提升至82.8%。
- NCDE改进：训练每步速度提升3个数量级。
- TabPack：默认配置匹敌调优方法，资源需求低。
- GSS：长地平线规划优于树搜索方法。

1. [From Fixed to Free Cameras: Calibration-Free View-Robust Vision-Language-Action Model](https://arxiv.org/abs/2607.05396v1)
   - 分类：Vision-Language-Action / View-Robust
   - 研究价值：值得精读
   - 关联：直接解决具身智能中相机视角变化问题，提出无需标定的视角鲁棒VLA模型。
   - 痛点：现有VLA策略依赖标定相机外参，在实际部署中相机位置经常改变，导致性能脆弱。需要一种无需标定、自动适应视角变化的策略。CamVLA通过解耦控制与视角几何，消除了对外参和深度的依赖，从而在任意相机配置下保持鲁棒性。
   - 方法：提出CamVLA，预测相机坐标系中的末端执行器动作（姿态无关的运动生成）和6-DOF手眼矩阵（视角几何标定），通过确定性几何变换组合得到基座动作。仅需单目RGB图像和任务指令，无需标定、深度或多视图。
   - 结果：在仿真和真实机器人上多种未见视角下一致提升成功率，项目页面显示显著优于依赖外参的基线，证明了无标定视角鲁棒VLA的有效性。

2. [Cortex: A Bidirectionally Aligned Embodied Agent Framework for Long-horizon Manipulation](https://arxiv.org/abs/2607.05377v1)
   - 分类：Embodied Agent / Long-horizon Manipulation
   - 研究价值：值得精读
   - 关联：解决长程操作中高层规划与底层执行之间的语义-运动学鸿沟。
   - 痛点：现有VLA模型因马尔可夫性难以处理长程任务，层次化方法存在规划语义与执行运动学之间的鸿沟，导致可执行性差。Cortex通过标准化技能原语和可执行性原则，自动标注大规模数据，缩小了规划与执行间的差距。
   - 方法：标准化32种规范技能原语，注入可执行性原则（代表性对象属性、轨迹可达性），自动标注4k小时视频和30小时仿真数据。采用事件平衡采样训练，结合上下文和技能约束推理工程，使通用VLM能够分解并执行长程任务。
   - 结果：在Libero-long和RoboTwin上分别提升3.1%和4.1%，并零样本完成未见过的真实长程任务（如多阶段化学实验），展现了VLM+VLA框架的泛化能力，而单独VLA微调无法实现。

3. [Fitted Occupancy-Ratio Evaluation without Bellman Completeness](https://arxiv.org/abs/2607.05375v1)
   - 分类：Offline RL / Off-Policy Evaluation
   - 研究价值：值得精读
   - 关联：提出无需Bellman完备性的离线策略评估方法，强化学习理论重要进展。
   - 痛点：离线策略评估中的占有率比估计通常需要Bellman完备性或投影算子稳定性，这些假设在实际中很难满足，导致评估偏差。FORE通过KL投影和伴随Bellman算子，仅需占有率比可实现性，放宽了关键假设。
   - 方法：提出FORE，利用伴随Bellman递归在KL散度下投影密度比，迭代求解单层密度比目标。关键是贝叶斯KL收缩性质保证了内置的收缩性，无需完备性假设。
   - 结果：理论上建立了占有率比可实现性作为充分条件，并给出有限样本后悔界。支持奖励加权、占有率加权Q评估和双重稳健估计，为离线评估提供了更实用的理论工具。

**主要痛点**
- 现有VLA策略依赖相机外参标定，视角变化时性能脆弱。
- 长程操作中高层规划与低层执行存在语义-运动学鸿沟。
- 离线策略评估需要Bellman完备性，实际难以满足。
- 强模型RL训练成本高，弱模型知识难以有效迁移。
- 世界模型在可变形物体上缺乏大规模真实数据支持。

**方法趋势**
- CamVLA：解耦相机坐标系动作与手眼矩阵，通过几何变换组合实现无标定视角鲁棒。
- Cortex：标准化技能原语并注入可执行性原则，自动标注数据训练VLM+VLA框架。
- FORE：通过KL投影和伴随Bellman递归估计占有率比，无需Bellman完备性。
- Direct-OPD：利用弱模型RL前后策略对数比值作为隐式奖励，蒸馏到强模型。
- Deform360：大规模多视角触觉数据采集与无标记3D跟踪，系统比较2D/3D世界模型。

**结果信号**

1. [LLM-as-a-Verifier: A General-Purpose Verification Framework](https://arxiv.org/abs/2607.05391v1)
   - 分类：Agent Verification
   - 研究价值：值得精读
   - 关联：通用验证框架，可迁移至多类Agent任务
   - 痛点：现有Agent验证方法通常输出离散分数（如0/1或1-5），无法区分解的细微正确性差异，且不能为强化学习提供密集反馈；同时需要额外训练特定分类器或依赖人工标注，成本高且泛化弱。本文旨在构建一种无需额外训练、通用且可扩展的验证框架，能够输出连续评分以支持细粒度验证、排序和RL信号。
   - 方法：核心创新在于将LLM的评分token logits分布进行期望计算，得到连续分数，替代传统离散打分。进一步提出三种验证扩展机制：(1) 评分粒度扩展，将单一评分分解为多个子标准（如正确性、完整性、效率）并分别评分聚合；(2) 重复评估扩展，多次采样评分取均值降低方差；(3) 标准分解扩展，将复杂评价拆解为简单子问题。同时设计基于连续分数的成本效率排序算法，快速从候选解中选出最优。
   - 结果：在Terminal-Bench V2、SWE-Bench Verified、RoboRewardBench、MedAgentBench四个基准上均取得SOTA，分别达86.5%、78.2%、87.4%、73.3%。此外，验证的连续信号可直接用于强化学习：在SAC和GRPO中替代稀疏奖励，在机器人和数学推理任务上提升样本效率。已扩展为Claude Code插件，供开发者监控Agent系统。

2. [CompactionRL: Reinforcement Learning with Context Compaction for Long-Horizon Agents](https://arxiv.org/abs/2607.05378v1)
   - 分类：Agent Reinforcement Learning
   - 研究价值：值得精读
   - 关联：解决长程Agent上下文窗口限制，提升RL效率
   - 痛点：长程Agent任务中，交互轨迹可能超过LLM的上下文窗口（如128K token），导致任务未完成即截断。现有上下文压缩方法（如摘要）与强化学习分离，无法联合优化任务执行和摘要生成，且摘要质量未针对RL目标进行调整，造成信息损失。
   - 方法：提出CompactionRL框架，在RL训练过程中同时优化任务执行和上下文压缩策略。关键设计包括：(1) token级损失归一化，使得压缩后的短轨迹与原始轨迹在损失尺度上可比；(2) 跨轨迹广义优势估计（GAE），允许压缩前后的优势函数正确传播；(3) 联合训练，LLM Agent学习何时需要压缩以及如何压缩，同时保持任务策略优化。
   - 结果：在SWE-bench Verified上，基于GLM-4.5-Air（106B-A30B）模型达到66.8% Pass@1，提升7.0个百分点；Terminal-Bench 2.0达到24.5%，提升3.1个百分点。基于GLM-4.7-Flash（30B-A3B）同样提升5.5和6.8个百分点。该框架已实际部署于GLM-5.2（750B-A40B）的RL训练管线中，验证了可扩展性。

3. [TREK: Distill to Explore, Reinforce to Refine](https://arxiv.org/abs/2607.05339v1)
   - 分类：RL Distillation
