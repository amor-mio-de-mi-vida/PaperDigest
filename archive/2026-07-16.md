1. [VideoRAE: Taming Video Foundation Models for Generative Modeling via Representation Autoencoders](https://arxiv.org/abs/2607.14088v1)
   - 分类：Video Generation / Representation Learning
   - 研究价值：值得精读
   - 关联：该方法利用冻结的视频基础模型进行高效生成建模，显著加速训练并提升重建质量，适用于视频生成与压缩。
   - 痛点：传统3D-VAE主要优化像素级重建，导致潜在表示语义和时空结构有限；而视频基础模型虽理解能力强，但其冻结表示能否转换为紧凑、可重建、生成友好的视频潜变量尚未解决。VideoRAE填补了这一空白，实现了从强语义表示到高效生成的转化。
   - 方法：提出VideoRAE，一种表示自编码器，利用冻结的视频基础编码器的多尺度层次特征，通过轻量级1D自注意力投影器压缩为连续潜变量或离散码本（多码本高维量化）。解码时通过局部-全局表示对齐目标与冻结教师模型，无需KL正则化即可保持语义。
   - 结果：在UCF-101上实现类到视频gFVD分别为40（AR）和93（DiT），优于现有方法，且收敛速度比竞争自编码器基线快约5倍。在2B规模文生视频研究中，用VideoRAE替换LTX-VAE在相近设置下实现更快收敛，验证了冻结VFM表示作为通用视频潜变量的有效性。

2. [TRACE: Turn-level Reward Assignment via Credit Estimation for Long-Horizon Agents](https://arxiv.org/abs/2607.13988v1)
   - 分类：Reinforcement Learning / Agentic Reward Assignment
   - 研究价值：值得精读
   - 关联：提供了一种无需额外批评器或过程监督的稠密信用分配方法，显著提升长程智能体工具使用能力，且与现有RL框架兼容。
   - 痛点：多轮智能体在长程任务（数十到数百次工具调用）中，仅使用最终结果奖励会面临稀疏和高方差问题，且失败轨迹中的有用动作被赋予与错误相同的负优势，导致信用分配困难。现有方法需额外批评器或过程标签，成本高。
   - 方法：提出TRACE，通过冻结参考模型获取黄金答案对数概率，转换为对数比率状态值，再使用时序差分（TD）变化导出每动作奖励。该单步对数比率TD分量能自动处理冗余工具调用，无需额外训练或冷启动SFT阶段。
   - 结果：在封闭网络BrowseComp-Plus上，将Qwen3-4B从7.2提升至35.6，Qwen3-30B-A3B从8.4提升至42.6；学习曲线显示更早提升和更快收敛。学习到的搜索行为可迁移至开放网络基准，证明该方法在纯RL设置下显著改善长程智能体性能。

3. [Do Agent Optimizers Compound? A Continual-Learning Evaluation on Terminal-Bench 2.0](https://arxiv.org/abs/2607.14004v1)
   - 分类：Agent Optimization / Continual Learning
   - 研究价值：值得精读
   - 关联：该研究揭示了智能体优化方法在持续学习场景下是否复合的关键条件，对实际部署中如何设计优化循环有重要指导意义。
   - 痛点：现有智能体优化方法通常报告一次性结果，但部署环境中任务不断涌现，优化需递归应用。关键问题是优化增益能否复合：首次优化后再次优化新任务时是否会侵蚀旧增益？传统评估未考虑此持续学习场景。
   - 方法：设计两阶段持续学习评估，在Terminal-Bench 2.0的困难任务上比较三种方法（GEPA、Meta Harness、RELAI-VCL）。所有方法在静态单阶段中提升基线，但引入新任务后表现分化。RELAI-VCL的唯一差异是在优化循环中内置了回归控制。
   - 结果：RELAI-VCL是唯一既正迁移到未见任务，又能在新任务加入后继续提升的方法，达到最高终身平均通过率76.4%（对比GEPA 66.0%、Meta Harness 64.6%、基线58.7%）。关键观察：仅当优化循环内置回归控制时，优化增益才能复合，避免泛化失败的捷径解。

**主要痛点**
- 视频生成中传统3D-VAE潜在表示语义不足，训练效率低。
- 长程智能体信用分配稀疏，失败轨迹中的有用动作被误判。
- 智能体优化方法在持续学习场景下增益不复合，缺乏回归控制。
- 模拟电路优化中传统方法泛化性差，标准RL探索效率低。
- ViT自注意力无法显式拒绝无关补丁，引入冗余信息。

**方法趋势**
- 冻结视频基础模型+轻量投影器+局部-全局对齐（VideoRAE）。
- 冻结参考模型对数概率转为TD奖励（TRACE）。
- 内置回归控制的持续学习评估框架。
- 从历史高性能状态重置回合（Lighthouse RL）。
- 基于查询-键相似度的显式补丁筛选（VisionScreen）。

**结果信号**

- TRACE：BrowseComp-Plus上Qwen3-4B从7.2提升至35.6。

   - 研究价值：值得精读

   - 痛点：多轮Agent在长序列任务（如复杂搜索）中，结果奖励稀疏且方差大，一个失败轨迹中可能包含许多有益动作，传统的仅基于结果奖励的强化学习方法无法有效区分优劣动作，导致学习效率低。现有方法通常需要训练额外的批评网络或过程奖励模型，增加复杂性和计算成本。
