1. [Hierarchical Denoising For Multi-Step Visual Reasoning](https://arxiv.org/abs/2607.15278v1)
   - 分类：Video Generation / Visual Reasoning
   - 研究价值：值得精读
   - 关联：提出层次化去噪框架，将多步视觉推理融入因果视频生成，显著提升推理一致性并降低延迟
   - 痛点：现有视频模型缺乏类人多步推理能力：流式自回归扩散模型效率高但推理能力有限，双向扩散可全局修正但推理成本高昂（密集帧级去噪）。两种范式在复杂逻辑推理任务中均难以兼顾逻辑一致性与低延迟流式生成，且缺乏专门的多步推理视频基准。
   - 方法：提出HDR（层次化去噪视觉推理），将视频潜变量组织成树状层次结构，粗去噪层保留不确定性假设用于全局规划，细去噪层逐步精化为具体视觉状态；并设计稀疏层次注意力模式（SHAP）降低时序注意力成本。构建分层多步视频推理基准，包含迷宫导航、汉诺塔等6个OOD任务。
   - 结果：在6个推理任务上，HDR相比流式自回归扩散基线成功率从34.22提升至60.29（相对增益76.2%），平均进度从76.00提升至89.56；推理延迟仅0.70秒/潜变量，比双向扩散快54.2倍；仅用2%训练数据即保留82.9%全数据性能（双向扩散仅52.0%），真实机器人实验验证物理交互潜力。

2. [MeanFlowNFT: Bringing Forward-Process RL to Average-Velocity Generators](https://arxiv.org/abs/2607.15273v1)
   - 分类：Diffusion Models / RL-based Generation
   - 研究价值：值得精读
   - 关联：将前向过程RL框架扩展到平均速度生成器，实现快速少步采样且性能超越多步RL调优扩散模型
   - 痛点：MeanFlow生成器通过预测平均速度实现快速少步采样，但现有前向过程RL方法（如DiffusionNFT）优化的是瞬时速度，无法直接用于平均速度模型；如何在保持MeanFlow快速生成优势的同时利用RL对齐人类偏好或任务目标是一个挑战。
   - 方法：提出MeanFlowNFT，利用MeanFlow恒等式构建诱导的瞬时速度预测器，将DiffusionNFT目标应用于该预测器，使奖励优化在MeanFlow上良定义；采样仍基于平均速度以保持少步生成优势。理论证明继承严格策略改进保证。在图像和视频生成上实验。
   - 结果：在SD3.5-M上MeanFlowNFT在8项指标中6项超越先前SOTA少步RL调优生成器；在Wan 2.1上4步MeanFlowNFT达到VBench 84.33，超越50步LongCat-Video RL（82.57），表明少步RL调优模型甚至可超越多步RL调优扩散模型。

3. [Mask-Aware Policy Gradients for Diffusion Language Models](https://arxiv.org/abs/2607.15200v1)
   - 分类：Diffusion Language Models / RL for LLM
   - 研究价值：值得精读
   - 关联：揭示MDLM生成中的掩码顺序决策，提出两阶段动作MDP并联合优化token与masking策略
   - 痛点：强化学习在掩码扩散语言模型（MDLM）中的应用受限于对数似然估计的难解性；现有方法仅建模token预测，忽略生成过程中位置掩码顺序的决策，导致策略梯度不完整，限制了MDLM在推理任务上的性能。
   - 方法：形式化MDLM生成的两阶段动作MDP：每步决定向掩码位置放置何种token以及哪些位置被重新掩码；推导出策略梯度自然分解为token项和masking项，并结合两者进行优化（Mask-Aware Policy Gradients）。
   - 结果：在数学推理GSM8K上达到87.1%，在代码生成MBPP上达到53.4%，均为MDLM领域SOTA；相比仅优化token项的基线，联合优化显著提升推理质量，验证了掩码顺序决策的重要性。

**主要痛点**
- 视频生成模型缺乏类人多步推理能力，流式与双向扩散难以兼顾效率与逻辑一致性
- 前向过程RL方法无法直接适配平均速度生成器，快速少步采样与RL对齐存在鸿沟
- 掩码扩散语言模型的RL优化忽略掩码顺序决策，导致策略梯度不完整

**方法趋势**
- 层次化潜变量树状结构+稀疏层次注意力（HDR）实现粗到细推理
- 利用MeanFlow恒等式构建瞬时速度预测器，将RL目标迁移至平均速度模型（MeanFlowNFT）
- 两阶段动作MDP分解token与masking策略梯度，联合优化（Mask-Aware Policy Gradients）

**结果信号**
- HDR在6个推理任务上成功率提升76%，推理速度提升54倍，数据效率高
- MeanFlowNFT在多数指标超越SOTA少步RL模型，4步采样性能超过50步RL调优模型
- Mask-Aware PG在GSM8K和MBPP上达到MDLM领域SOTA，验证掩码顺序决策重要性

### 高效训练与推理

1. [RoboTTT: Context Scaling for Robot Policies](https://arxiv.org/abs/2607.15275v1)
   - 分类：Robot Policy / Test-Time Training
   - 研究价值：值得精读
   - 关联：首次将上下文扩展到8K时间步，实现机器人策略的一次性上下文模仿学习，显著提升长时任务性能。
   - 痛点：现有机器人策略模型仅使用单步或短历史上下文，无法有效处理长时任务和多阶段操作，且推理延迟随上下文增长。本文解决如何在不增加推理延迟的情况下，将视觉运动上下文扩展到数千时间步，从而解锁一次性模仿学习、在线策略改进和鲁棒性等新能力。
   - 方法：提出RoboTTT，将测试时训练（TTT）集成到视觉-语言-动作策略中，通过梯度下降在训练和推理时更新快速权重作为循环状态，压缩历史信息。结合序列动作强制和截断反向传播来扩展训练上下文长度，实现8K时间步上下文，推理时不增加延迟。
   - 结果：在真实机器人操作任务中，整体性能比单步基线提升87%，完整完成5分钟10阶段装配任务（基线从未完成）。8K上下文预训练比1K上下文性能提升62%，证明上下文长度作为机器人基础模型新扩展轴。

2. [MeanFlowNFT: Bringing Forward-Process RL to Average-Velocity Generators](https://arxiv.org/abs/2607.15273v1)
   - 分类：Image/Video Generation / RL for Diffusion
   - 研究价值：值得精读
   - 关联：将前向过程RL扩展到平均速度生成器，实现快速几步采样对齐，优于多步RL方法。
   - 痛点：现有RL对齐扩散/流模型的方法需要反向过程轨迹或似然估计，效率低。DiffusionNFT虽高效但仅适用于瞬时速度生成器，不适用于平均速度的MeanFlow生成器。需要一种方法在保持MeanFlow快速采样的同时进行有效RL对齐。
   - 方法：提出MeanFlowNFT，利用MeanFlow恒等式连接平均速度和瞬时速度，构建诱导的瞬时速度预测器，在其上应用DiffusionNFT目标进行奖励优化。采样仍基于平均速度，保留几步生成优势。理论证明具有严格的策略改进保证。
   - 结果：在图像和视频生成上持续改进基线，在SD3.5-M上8个指标中6个超越先前最先进RL调优的几步生成器，甚至超越多步RL调优扩散（如Wan 2.1上4步MeanFlowNFT达到VBench 84.33，超越50步LongCat-Video RL的82.57）。

3. [T^2MLR: Transformer with Temporal Middle-Layer Recurrence](https://arxiv.org/abs/2607.15178v1)
   - 分类：Transformer Architecture / Reasoning
   - 研究价值：值得精读
   - 关联：通过中间层循环增强Transformer推理能力，无需从头预训练即可显著提升数学推理。
   - 痛点：Transformer自回归解码限制了中间推理状态的持久性，每次压缩为下一个token导致推理深度受限。现有循环方法需要所有层的循环，计算成本高且需从头训练。需要一种轻量级方法提升推理能力而不增加大量开销。
   - 方法：提出T2MLR，在Transformer中引入时间中间层循环：将前一个token的中间层表示缓存并注入到当前token的早期层，使抽象中间计算在解码步骤间持久化。仅需对局部中间层块（少于20%网络）应用循环，可集成到现有预训练模型。
   - 结果：在自然语言预训练和多跳推理微调中一致超过同规模Transformer基线。将循环通路改装到已有的1.7B预训练模型并简短微调，极大提升数学推理性能，证明中间层循环是提升推理的有效且实用的途径。

**主要痛点**
- 机器人策略上下文短，无法处理长时多阶段任务
- 生成模型RL对齐不适用于平均速度生成器
- Transformer推理中间状态难以持久化
- 视频模型缺乏多步推理能力
- 搜索agent陷入重复循环

**方法趋势**
- 测试时训练集成到机器人策略中
- 利用恒等式构建瞬时速度预测器进行RL
- 中间层时间循环持久化推理状态
- 层次化潜变量粗到细去噪
- 外部化搜索状态为证据图

**结果信号**
- 机器人任务性能提升87%，完成10阶段任务
- 几步采样超越多步RL方法
- 推理能力显著提升，可改装到现有模型
- 成功率提升76%，速度提升54倍
- 全面领先搜索基线

1. [RoboTTT: Context Scaling for Robot Policies](https://arxiv.org/abs/2607.15275v1)
   - 分类：Robot Policy / Test-Time Training
   - 研究价值：值得精读
   - 关联：将测试时训练引入机器人基础模型，实现超长上下文，为机器人策略提供新的缩放维度。
   - 痛点：现有机器人基础模型通常只使用单步或短历史视觉运动上下文，无法利用长序列信息进行策略学习，导致在长时任务、从人类演示中一次模仿、在线策略改进等能力上存在明显不足。具体而言，随着任务时长增加，模型需要更长的上下文来追踪进度、处理扰动，但现有方法在推理时难以扩展上下文长度而不爆炸延迟。
   - 方法：提出RoboTTT，将测试时训练（Test-Time Training）集成到视觉-语言-动作策略中，形成循环状态由快速权重（梯度下降更新的参数）构成的序列模型。训练时采用序列动作强制（sequence action forcing）与截断反向传播通过时间（TBPTT）结合，实现8K时间步的上下文缩放。推理时通过在线梯度更新压缩历史，实现长上下文条件化而延迟不变。
   - 结果：在真实机器人操作任务上，RoboTTT相比单步上下文基线整体性能提升87%，且能完整完成5分钟10阶段组装任务（基线从未成功）。8K上下文预训练模型相比1K提升62%，首次观察到闭环性能随预训练上下文长度稳定增长。此外，8K上下文推理延迟仅0.70秒每潜变量，比双向扩散模型快54.2倍。

2. [Scaling Behavior Foundation Model for Humanoid Robots](https://arxiv.org/abs/2607.15163v1)
   - 分类：Humanoid Control / Behavior Foundation Model
   - 研究价值：值得精读
   - 关联：系统研究人形机器人行为基础模型的缩放规律，提出协调学习范式、数据多样性和模型架构的关键要素。
   - 痛点：人形机器人控制需要全身协调、实时响应和跨场景泛化，行为基础模型（BFM）有潜力但缩放规律不明。现有工作在缩放时缺乏对学习范式（如运动追踪）、行为数据（在线采集与参考动作多样性）和模型架构（表达能力与可扩展性）三者协调的系统性指导，导致性能提升不稳定且泛化受限。
   - 方法：提出协调缩放配方：(1) 将多样化人形控制问题重新形式化为全局坐标系下整合全身行为的运动追踪学习范式；(2) 策略性地协同在线策略收集数量与参考动作多样性；(3) 设计Humanoid Transformer架构，具有表达性和可扩展性，能自然涌现结构化行为表示。通过仿真和大规模真实世界部署验证。
   - 结果：相比现有控制器，在测试集上局部模式均值每关键点位置误差（MPKPE）降低超10%，全局模式降低82%。实验证明该缩放配方能使BFM在控制保真度和任务泛化上取得显著提升，为人形机器人通用控制器提供了可扩展基础。

3. [BadWAM: When World-Action Models Dream Right but Act Wrong](https://arxiv.org/abs/2607.15207v1)
   - 分类：Adversarial Attack / World-Action Model
   - 研究价值：值得精读
   - 关联：首次系统性揭示世界动作模型（WAM）的对抗脆弱性，提出两类攻击并分析其安全影响。
   - 痛点：世界动作模型（WAM）被认为通过联合预测动作和未来世界状态提供鲁棒性和安全性，但这一假设未经严格检验。实际中，WAM可能对微小视觉扰动敏感，导致想象正确但执行错误，这种“世界-动作漂移”攻击尚未被定义和评估，存在严重安全隐患。
   - 方法：提出BadWAM统一框架，定义两类攻击：(1) 动作唯一对抗攻击，直接驱动任务失败动作；(2) 想象保持对抗攻击，在保持未来预测接近干净想象的同时诱导有害动作。攻击使用小视觉扰动，通过优化实现攻击强度和隐蔽性的权衡。评估不同WAM变体。
   - 结果：动作唯一攻击将任务成功率从96.5%降至43.1%；想象保持攻击在保持未来想象漂移较小的同时仍显著降低成功率，暴露了WAM特有的脆弱性。实验表明WAM的安全耦合是可破坏的，需要防御机制。

**主要痛点**
- 机器人策略上下文短，难以执行长时任务和在线适应。
- 人形机器人行为基础模型缩放缺乏系统指导。
- 世界动作模型存在对抗脆弱性，想象与动作可能脱钩。
- 掩码扩散语言模型RL优化未考虑掩码顺序。
- 同策略蒸馏传递了与推理无关的噪声。

**方法趋势**
- 测试时训练和序列动作强制。
- 协调学习范式、数据多样性和模型架构。
- 对抗攻击优化视觉扰动。
- 两阶段动作MDP和政策梯度分解。
- Delta信号蒸馏。

**结果信号**
- 机器人策略性能提升87%，8K上下文比1K提升62%。
- 人形机器人MPKPE降低82%，全局模式验证。
- WAM成功率从96.5%降至43.1%。
- MDLM在GSM8K达87.1%。
- 蒸馏超越传统方法且步骤更少。

**原始摘要**

In this work, we investigate to what extent LLM estimates adhere to this self-consistency principle. Together, these findings suggest that models possess relevant subpopulation knowledge but do not reliably propagate it into aggregate estimates. This gap establishes statistical self-consistency as an unsaturated, reference-free criterion for evaluating LLMs.
